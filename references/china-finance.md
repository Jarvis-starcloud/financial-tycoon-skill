## 一、中国金融监管体系

### 一委一行一局两会

```
国务院金融稳定发展委员会（金融委）
        |
        ├── 中国人民银行（央行）
        |     └── 货币政策、宏观审慎、金融稳定
        |
        ├── 国家金融监督管理总局
        |     └── 银行、保险、非银监管
        |
        └── 中国证监会
              └── 证券、期货、上市公司监管
```

### 监管改革历程

| 年份 | 事件 |
|------|------|
| 1992 | 证监会成立 |
| 1998 | 保监会成立 |
| 2003 | 银监会成立 |
| 2017 | 金融委成立 |
| 2018 | 银保监会合并 |
| 2023 | 金融监管总局成立 |

## 二、A股市场特点

### 市场结构

```
上海证券交易所
    ├── 主板（蓝筹）
    └── 科创板（硬科技）

深圳证券交易所
    ├── 主板（蓝筹+中小）
    └── 创业板（成长）

北京证券交易所
    └── 专精特新
```

### 交易制度

| 制度 | 内容 |
|------|------|
| 涨跌停 | 主板±10%，创业板/科创板±20% |
| T+1 | 当日买入次日可卖 |
| 竞价制度 | 集合竞价+连续竞价 |
| 最小单位 | 100股（1手） |
| 注册制 | 全面实施 |

### 投资者结构

| 类型 | 特点 |
|------|------|
| 散户 | 占比高，换手率高 |
| 公募基金 | 主要机构投资者 |
| 私募基金 | 策略多样 |
| 保险资金 | 长期稳健 |
| 社保养老金 | 超长期 |
| 外资 | 北向资金关注度高 |

### A股特有现象

| 现象 | 描述 |
|------|------|
| 政策市 | 政策对市场影响大 |
| 炒小炒新 | 偏好小盘、新股 |
| 板块轮动 | 热点快速切换 |
| 年末效应 | 机构年底调仓 |
| 春节前效应 | 节前通常偏强 |

## 四、沪深港通

### 机制

```
沪股通：境外投资者 → 上交所股票
港股通（沪）：内地投资者 → 港交所股票

深股通：境外投资者 → 深交所股票
港股通（深）：内地投资者 → 港交所股票
```

### 北向资金

| 特点 | 描述 |
|------|------|
| 来源 | 境外投资者通过港交所 |
| 额度 | 每日520亿元 |
| 数据 | 每日公布净买入 |
| 意义 | 外资风向标 |

**北向资金偏好**：
- 核心资产
- 大消费
- 龙头白马
- 高ROE

### 南向资金

内地投资者买港股的资金

| 融资余额下降 | 杠杆资金看空 |

| 融券余额 | 做空力量 |

### 风险

- 强制平仓风险
- 杠杆放大亏损
- 标的限制

## 六、公募基金

### 基金类型

| 类型 | 投资范围 | 风险 |
|------|----------|------|
| 货币基金 | 货币市场工具 | 低 |
| 债券基金 | 债券为主 | 中低 |
| 混合基金 | 股债混合 | 中 |
| 股票基金 | 股票为主 | 高 |
| 指数基金 | 跟踪指数 | 视标的 |
| QDII | 境外投资 | 高+汇率 |
| REITs | 基础设施 | 中 |

### 基金指标

| 指标 | 含义 |
|------|------|
| 净值 | 单位资产价值 |
| 规模 | 总资产 |
| 费率 | 管理费+托管费 |
| 夏普比率 | 风险调整收益 |
| 最大回撤 | 历史最大亏损 |
| Alpha | 超额收益能力 |

### 基金投资策略

- **定投**：定期定额，平滑成本
- **核心卫星**：宽基+主题
- **FOF**：基金的基金
- **目标日期**：随年龄调整

## 八、房地产与金融

### 房地产对金融的重要性

- 银行贷款大量投向房地产
- 房产是最主要抵押品
- 地方政府土地财政依赖

### 房地产政策

| 政策 | 目的 |
|------|------|
| 限购 | 需求端调控 |
| 限贷 | 杠杆控制 |
| 限售 | 投机抑制 |
| 限价 | 价格管控 |
| 三道红线 | 企业杠杆控制 |

### 三道红线

| 红线 | 标准 |
|------|------|
| 剔除预收账款后资产负债率 | ≤70% |
| 净负债率 | ≤100% |
| 现金短债比 | ≥1 |

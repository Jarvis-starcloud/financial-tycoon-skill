## 一、货币的本质

### 货币的职能

| 职能 | 描述 |
|------|------|
| 价值尺度 | 衡量商品价值 |
| 流通手段 | 交换媒介 |
| 贮藏手段 | 保存价值 |
| 支付手段 | 延期支付 |
| 世界货币 | 国际结算 |

### 货币形式演变

```
实物货币 → 金属货币 → 纸币 → 电子货币 → 数字货币
```

### 货币层次

| 层次 | 定义 | 流动性 |
|------|------|--------|
| M0 | 流通中现金 | 最高 |
| M1 | M0 + 活期存款 | 高 |
| M2 | M1 + 定期存款 + 储蓄存款 | 中 |
| M3 | M2 + 其他金融资产 | 低 |

**中国重点关注M2**：
- 广义货币供应量
- 央行货币政策中间目标

## 二、货币创造机制

### 中央银行创造基础货币

**基础货币（B）**：
```
B = 现金(C) + 银行准备金(R)
```

**基础货币来源**：
| 渠道 | 描述 |
|------|------|
| 外汇占款 | 央行买入外汇投放人民币 |
| 对金融机构债权 | MLF、SLF等 |
| 对政府债权 | 买入国债 |
| 其他 | 再贷款等 |

### 商业银行创造存款货币

**货币乘数**：
```
m = M2 / B ≈ 1 / 法定准备金率
```

**实际乘数更低**，因为：
- 超额准备金
- 现金漏损
- 定期存款比例

### 货币创造示例

```
央行投放100元基础货币
法定准备金率10%

银行A：收100元存款，留10元准备金，贷出90元
银行B：收90元存款，留9元准备金，贷出81元
银行C：收81元存款，留8.1元准备金，贷出72.9元
...

最终创造存款 = 100 × (1/10%) = 1000元
```

## 四、货币政策工具

### 传统工具

| 工具 | 作用机制 |
|------|----------|
| 存款准备金率 | 调节货币乘数 |

## 五、利率体系

### 利率决定理论

| 理论 | 核心观点 |
|------|----------|
| 古典理论 | 储蓄与投资决定 |
| 流动性偏好 | 货币供求决定 |
| 可贷资金理论 | 综合实际与货币因素 |

### 中国利率体系

```
政策利率
    |
    ├── OMO利率（短端）
    |
    └── MLF利率（中端）
            |
            └── LPR（贷款基准）
                    |
                    ├── 1年期LPR → 一般贷款
                    |
                    └── 5年期LPR → 房贷
```

### 利率传导机制

```
政策利率 → 货币市场利率 → 债券利率 → 贷款利率 → 实体经济
```

## 六、信用与信贷

### 信用类型

| 类型 | 描述 |
|------|------|
| 商业信用 | 企业间赊销 |
| 银行信用 | 银行贷款 |
| 国家信用 | 国债 |
| 消费信用 | 消费贷款 |

### 信贷市场

**贷款分类**：
| 维度 | 分类 |
|------|------|
| 期限 | 短期/中期/长期 |
| 用途 | 经营/固定资产/消费 |
| 担保 | 信用/抵押/质押/保证 |

### 信贷周期

```
扩张期：信贷宽松、资产价格上涨
    ↓
过热期：杠杆率上升、风险累积
    ↓
收缩期：信贷紧缩、违约上升
    ↓
出清期：去杠杆、风险释放
```

## 七、银行业务

### 资产负债表

| 资产 | 负债 + 权益 |
|------|-------------|
| 贷款 | 存款 |
| 债券投资 | 同业负债 |
| 同业资产 | 发行债券 |
| 现金及准备金 | 股东权益 |

### 盈利模式

```
净利息收入 = 利息收入 - 利息支出
净息差(NIM) = 净利息收入 / 生息资产
```

### 风险管理

| 风险类型 | 管理方法 |
|----------|----------|
| 信用风险 | 贷款审批、拨备 |
| 流动性风险 | 流动性比例、LCR |
| 市场风险 | VaR、对冲 |
| 操作风险 | 内控、合规 |

### 监管指标

| 指标 | 要求 |
|------|------|
| 资本充足率 | ≥10.5% |
| 拨备覆盖率 | ≥150% |
| 流动性覆盖率(LCR) | ≥100% |
| 不良贷款率 | 越低越好 |

## 九、汇率与外汇

### 汇率制度

| 制度 | 特点 |
|------|------|
| 固定汇率 | 央行承诺维持 |
| 浮动汇率 | 市场决定 |
| 管理浮动 | 有管理的浮动 |
| 联系汇率 | 与某货币挂钩 |

### 人民币汇率

- **中间价**：每日央行公布
- **即期汇率**：日内波动±2%
- **离岸汇率(CNH)**：香港市场

### 汇率影响因素

| 因素 | 机制 |
|------|------|
| 利差 | 利率平价 |
| 通胀差 | 购买力平价 |
| 国际收支 | 供求关系 |
| 经济增长 | 长期趋势 |
| 央行干预 | 直接影响 |
| 市场情绪 | 短期波动 |

### 外汇储备

**中国外汇储备**：约3万亿美元

**作用**：
- 稳定汇率
- 应对外部冲击
- 国际支付能力

## 十、金融危机

### 危机类型

| 类型 | 描述 |
|------|------|
| 银行危机 | 银行挤兑、破产 |
| 货币危机 | 汇率崩溃 |
| 债务危机 | 主权债务违约 |
| 系统性危机 | 多种危机叠加 |

### 金融危机周期

```
信贷扩张 → 资产泡沫 → 泡沫破裂 → 信贷收缩 → 经济衰退 → 政策刺激 → 复苏
```

### 历史金融危机

| 年份 | 危机 | 影响 |
|------|------|------|
| 1929 | 大萧条 | 全球 |
| 1997 | 亚洲金融危机 | 亚洲 |
| 2008 | 次贷危机 | 全球 |
| 2010 | 欧债危机 | 欧洲 |

### 危机应对工具

| 工具 | 描述 |
|------|------|
| 降息 | 降低融资成本 |
| QE | 注入流动性 |
| 财政刺激 | 政府支出 |
| 银行救助 | 注资、担保 |
| 资本管制 | 限制资金外流 |

## 十一、货币政策与市场

### 对股市影响

| 政策 | 股市影响 |
|------|----------|
| 降息 | 利好（估值提升） |
| 加息 | 利空（估值压缩） |
| 降准 | 利好（流动性） |
| QE | 利好（资产价格） |

### 对债市影响

| 政策 | 债市影响 |
|------|----------|
| 降息 | 利好（价格上涨） |
| 加息 | 利空（价格下跌） |
| 宽松预期 | 收益率下行 |
| 紧缩预期 | 收益率上行 |

### 对汇率影响

| 政策 | 汇率影响 |
|------|----------|
| 降息 | 本币贬值压力 |
| 加息 | 本币升值压力 |
| QE | 本币贬值 |
| QT | 本币升值 |

## 🚨 核心风控铁律（最重要）

> ⚠️ **以下规则是交易的生命线，任何情况下必须严格遵守**

### 硬性风控参数

| 风控维度 | 保守档 | 标准档 | 激进档 | **强制上限** |
|----------|--------|--------|--------|--------------|
| **单笔最大亏损** | ≤0.5% | ≤1% | ≤1.5% | **≤2%** |
| **单日最大亏损** | ≤2% | ≤3% | ≤4% | **≤5%** |
| **单品种最大仓位** | ≤15% | ≤20% | ≤25% | **≤30%** |
| **总杠杆上限** | 1x | 1.5x | 2x | **固定，禁止动态放大** |

### 强制熔断机制

```
🔴 触发任一条件 → 强制暂停交易

1. 连续亏损 5 次 → 暂停交易至少 24 小时
2. 单日亏损触及 5% → 当日停止所有交易
3. 周亏损超过 10% → 暂停一周，复盘后再交易
4. 月亏损超过 15% → 暂停两周，全面检视策略
```

### 风控计算公式

**单笔最大可亏损金额**：
```
最大亏损金额 = 总本金 × 单笔亏损比例(0.5%~2%)

例：100万本金，1%风控
最大亏损 = 1,000,000 × 1% = 10,000元
```

**止损价计算**：
```
多头止损价 = 入场价 × (1 - 最大亏损金额 / 持仓市值)
空头止损价 = 入场价 × (1 + 最大亏损金额 / 持仓市值)
```

**仓位计算（凯利公式简化版）**：
```
建议仓位 = (最大亏损金额 / 止损幅度) / 总本金

例：100万本金，最大亏损1万，止损5%
仓位 = (10,000 / 5%) / 1,000,000 = 20%
```

### 风控执行清单（每笔交易前必检）

```
□ 1. 确认未处于强制暂停状态
□ 2. 检查连续亏损次数（≥5次禁止交易）
□ 3. 检查今日已亏损比例（≥5%禁止交易）
□ 4. 计算本笔最大可亏损金额
□ 5. 确定止损位置和止损价格
□ 6. 根据止损幅度计算合理仓位
□ 7. 检查单品种仓位是否超限（≤30%）
□ 8. 设置好止损单（必须在入场同时设置）
```

### 连续亏损追踪与应对

| 连续亏损次数 | 状态 | 应对措施 |
|--------------|------|----------|
| 1-2次 | 🟢 正常 | 继续执行，检视是否符合计划 |
| 3次 | 🟡 警戒 | 降低仓位50%，增加观察时间 |
| 4次 | 🟠 预警 | 降低仓位75%，只做最有把握的机会 |
| **5次** | 🔴 **熔断** | **强制暂停交易至少24小时** |

### 恢复交易条件

连续亏损5次暂停后，需满足以下条件才能恢复：
1. ✅ 已完成详细复盘，找出亏损原因
2. ✅ 已制定改进计划
3. ✅ 等待时间已满24小时
4. ✅ 心态已调整（非报复性交易）
5. ✅ 恢复后首周仓位降至正常的50%

## 一、风险的本质

### 风险定义

- **学术定义**：收益的不确定性（波动）
- **实践定义**：永久性资本损失的可能性

### 风险类型

| 类型 | 描述 | 可否分散 |
|------|------|----------|
| 系统风险 | 市场整体风险 | 不可分散 |
| 非系统风险 | 个股/行业风险 | 可分散 |

### 风险度量

| 指标 | 公式/描述 |
|------|-----------|
| 标准差 | 收益率波动程度 |
| Beta | 相对市场的波动 |
| VaR | 特定置信度下最大损失 |
| 最大回撤 | 历史最大亏损幅度 |
| 夏普比率 | (收益-无风险)/波动 |

## 二、仓位管理

### 总仓位控制

| 市场状态 | 建议仓位 | 依据 |
|----------|----------|------|
| 极度恐惧 | 60-80% | 恐惧贪婪<25 |
| 恐惧 | 40-60% | 恐惧贪婪25-45 |
| 中性 | 30-50% | 恐惧贪婪45-55 |
| 贪婪 | 20-30% | 恐惧贪婪55-75 |
| 极度贪婪 | 0-20% | 恐惧贪婪>75 |

### 单票仓位控制

**普通投资者**：
- 单票上限：10%-15%
- 核心持仓：5只以内
- 卫星持仓：可适度分散

**凯利公式**：
```
f* = (bp - q) / b

其中：
- f*：最优仓位比例
- b：赔率（预期收益/本金）
- p：获胜概率
- q：失败概率（1-p）
```

### 加仓策略

| 策略 | 方法 | 适用 |
|------|------|------|
| 金字塔加仓 | 越跌买越多 | 左侧交易 |
| 倒金字塔加仓 | 越涨买越多 | 右侧交易 |
| 等额加仓 | 每次等量 | 定投 |
| 目标仓位法 | 偏离后再平衡 | 长期配置 |

## 三、止损策略

### 止损的意义

- 控制单笔最大损失
- 保护整体资本
- 减少情绪影响

### 止损方法

| 方法 | 描述 | 设置 |
|------|------|------|
| 固定比例止损 | 亏损固定比例离场 | 5%-10% |
| 技术位止损 | 破关键支撑离场 | 支撑下方2%-3% |
| 移动止损 | 随盈利提高止损位 | 保护浮盈 |
| 时间止损 | 超时未达预期离场 | 1-3个月 |
| 基本面止损 | 逻辑破坏离场 | 核心假设失效 |

### 止损原则

1. **事前设定**：买入前就定好
2. **严格执行**：到位就走，不抱幻想
3. **不频繁调整**：除非有新信息
4. **接受小亏**：小亏防大亏

|------|------|

## 五、资产配置理论

### 现代投资组合理论（MPT）

**核心思想**：
- 分散化可降低风险
- 存在有效前沿
- 最优组合取决于风险偏好

**有效前沿**：
```
在相同风险下收益最高
或
在相同收益下风险最低
```

### 资本资产定价模型（CAPM）

```
E(Ri) = Rf + βi × (E(Rm) - Rf)

其中：
- E(Ri)：资产预期收益
- Rf：无风险利率
- βi：资产Beta
- E(Rm)：市场预期收益
```

### 相关性的作用

| 相关系数 | 分散效果 |
|----------|----------|
| +1 | 无分散效果 |
| 0 | 部分分散 |
| -1 | 完全对冲 |

## 六、资产配置策略

### 战略资产配置（SAA）

- 长期目标配置比例
- 基于风险承受能力
- 定期再平衡

### 战术资产配置（TAA）

- 短期偏离战略配置
- 捕捉市场机会
- 控制偏离幅度

### 核心-卫星策略

```
组合 = 核心（70%） + 卫星（30%）
```

| 部分 | 内容 | 特点 |
|------|------|------|
| 核心 | 宽基指数、蓝筹 | 稳定、低成本 |
| 卫星 | 主题、个股 | 增强收益 |

### 再平衡

**时间触发**：
- 每季度/半年/年度

**阈值触发**：
- 偏离5%-10%时

## 十、行为风险管理

### 常见行为偏误

| 偏误 | 表现 | 对策 |
|------|------|------|
| 过度自信 | 交易过频 | 降低交易频率 |

3. **定期复盘**：检视执行情况
4. **控制情绪**：市场极端时尤其重要

## 十一、风险收益评估

### 夏普比率

```
Sharpe = (Rp - Rf) / σp

其中：
- Rp：组合收益
- Rf：无风险收益
- σp：组合波动率
```

| 夏普比率 | 评价 |
|----------|------|
| <0 | 差 |
| 0-0.5 | 一般 |
| 0.5-1 | 良好 |
| >1 | 优秀 |

### 最大回撤

```
最大回撤 = (峰值 - 谷值) / 峰值
```

可接受标准因人而异，一般<20%

### 卡玛比率

```
Calmar = 年化收益 / 最大回撤
```

衡量收益与最大风险的关系

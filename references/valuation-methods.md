## 一、估值方法分类

### 相对估值法

- 与可比公司/历史对比
- 简单直观
- 依赖可比性假设

### 绝对估值法

- 基于内在价值计算
- 理论严谨
- 依赖假设参数

---

## 二、PE估值（市盈率）

### 基本公式

```
PE = 市值 / 净利润 = 股价 / 每股收益(EPS)
```

### PE类型

| 类型 | 定义 | 适用 |
|------|------|------|
| 静态PE | 市值/上年净利 | 年报已出 |
| 滚动PE(TTM) | 市值/近四季净利 | 最常用 |
| 动态PE | 市值/预测净利 | 前瞻估值 |

### PE估值应用

**适用**：
- 盈利稳定的成熟公司
- 周期性不强的行业

**不适用**：
- 亏损公司
- 周期股底部
- 高速成长股

### PE参考区间

| PE水平 | 一般判断 |
|--------|----------|
| <10 | 低估或有问题 |
| 10-20 | 合理区间 |
| 20-40 | 需要成长性支撑 |
| >40 | 高估或高成长 |

**注意**：不同行业PE中枢差异大

---

## 三、PB估值（市净率）

### 基本公式

```
PB = 市值 / 净资产 = 股价 / 每股净资产
```

### 适用场景

| 行业 | 原因 |
|------|------|
| 银行 | 资产负债表为核心 |
| 保险 | 内含价值相关 |
| 地产 | 土地资产为核心 |
| 周期股 | 利润波动大 |

### PB与ROE关系

```
合理PB = ROE / 要求回报率
```

例如：ROE=15%，要求回报率10%
→ 合理PB = 15%/10% = 1.5倍

### 破净股分析

PB<1的可能原因：
1. 资产质量差（减值风险）
2. ROE过低
3. 行业前景悲观
4. 市场错杀（机会）

---

## 四、PS估值（市销率）

### 基本公式

```
PS = 市值 / 营业收入 = 股价 / 每股销售额
```

### 适用场景

- 亏损但有收入的公司
- 高成长期的互联网公司
- SaaS等订阅制企业

### 局限性

- 不考虑盈利能力
- 不同毛利率公司不可比

---

## 五、PEG估值

### 基本公式

```
PEG = PE / 净利润增长率(%)

例如：PE=30，增长率30% → PEG=1
```

### 判断标准

| PEG | 判断 |
|-----|------|
| <0.5 | 明显低估 |
| 0.5-1 | 相对低估 |
| 1 | 合理 |
| 1-2 | 相对高估 |
| >2 | 明显高估 |

### 局限性

- 增长率难以准确预测
- 不适用于周期股
- 不适用于低增长公司

## 六、EV/EBITDA估值

### 核心概念

**EV（企业价值）**：
```
EV = 市值 + 净债务
   = 市值 + 有息负债 - 现金
```

**EBITDA**：
```
EBITDA = 息税折旧摊销前利润
       = 营业利润 + 折旧 + 摊销
```

### 公式

```
EV/EBITDA = 企业价值 / EBITDA
```

### 优势

1. 剔除资本结构影响
2. 剔除折旧政策差异
3. 跨国比较更公允

### 适用场景

- 高杠杆行业
- 重资产行业
- 跨国并购估值

## 七、DCF估值（现金流折现）

### 核心公式

```
内在价值 = Σ(FCF_t / (1+WACC)^t) + 终值/(1+WACC)^n
```

### 自由现金流（FCF）

```
FCF = 经营现金流 - 资本支出
    = EBIT×(1-税率) + 折旧 - 资本支出 - 营运资本增加
```

### WACC计算

```
WACC = E/(E+D)×Ke + D/(E+D)×Kd×(1-T)

其中：
- E：股权价值
- D：债务价值
- Ke：股权成本（CAPM：Rf + β×(Rm-Rf)）
- Kd：债务成本
- T：所得税率
```

### 终值计算

**永续增长法**：
```
终值 = FCF_n×(1+g) / (WACC-g)
```
g通常取2%-3%（接近GDP增速）

**退出乘数法**：
```
终值 = EBITDA_n × 退出EV/EBITDA倍数
```

### DCF敏感性分析

| WACC变化 | 终值变化 | 估值影响 |
|----------|----------|----------|
| +1% | - | -15%~-25% |
| -1% | - | +15%~+25% |
| - | +1%增长率 | +10%~+20% |

### DCF局限性

- 对假设高度敏感
- 预测长期现金流困难
- 不适用于亏损公司

## 八、股息折现模型（DDM）

### Gordon模型

```
P = D1 / (r - g)

其中：
- D1：下期股息
- r：要求回报率
- g：股息增长率
```

### 适用场景

- 稳定分红的成熟公司
- 公用事业、REITs

### 局限性

- 要求g < r
- 不适用于不分红公司

---

## 九、分部加总估值（SOTP）

### 基本思路

```
总估值 = Σ(各业务板块估值)
```

### 适用场景

- 多元化集团
- 各业务差异大
- 分拆价值分析

### 方法

1. 拆分各业务收入/利润
2. 分别选择可比公司
3. 分别估值
4. 加总得到总估值

## 十、A股特色估值

### 茅指数

- 核心资产代表
- 消费白马股
- 估值中枢较高

### 宁组合

- 新能源赛道
- 高成长高估值
- PEG估值更合适

### 中特估

- 央国企重估
- PB修复逻辑
- 政策驱动

### 壳价值

- 上市资格溢价
- 注册制后逐步消失

## 十一、估值的艺术

### 估值不是精确科学

> "模糊的正确胜过精确的错误" —— 巴菲特

### 安全边际

```
买入价格 = 内在价值 × (1 - 安全边际)
```

建议安全边际：20%-40%

### 估值的相对性

- 相对于自身历史
- 相对于同行业
- 相对于市场整体
- 相对于无风险利率

### 估值陷阱

| 陷阱 | 表现 |
|------|------|
| 价值陷阱 | 低PE但基本面恶化 |
| 成长陷阱 | 高增长但无法持续 |
| 概念陷阱 | 故事好但业绩差 |

## 十二、实用估值工作流

### 第一步：选择合适方法

| 公司类型 | 推荐方法 |
|----------|----------|
| 稳定盈利 | PE、DCF |
| 高成长 | PEG、PS |
| 重资产 | PB、EV/EBITDA |
| 亏损期 | PS、EV/Sales |
| 高股息 | DDM、股息率 |

### 第二步：多方法交叉验证

至少用2-3种方法

### 第三步：敏感性分析

测试关键假设变化影响

### 第四步：得出估值区间

不追求点估值，而是区间

### 第五步：考虑安全边际

留出错误空间

## 一、产业分析的意义

### 为什么要分析行业

1. **自上而下选股**：好行业里选好公司
2. **理解公司处境**：竞争环境决定盈利能力
3. **把握投资时机**：行业景气周期
4. **识别风险**：行业性风险往往难以分散

### 行业与个股的关系

> "即使最好的公司，在夕阳行业也难以长期跑赢"

| 组合 | 预期 |
|------|------|
| 好行业 + 好公司 | 最佳 |
| 好行业 + 差公司 | 中等 |
| 差行业 + 好公司 | 困难 |
| 差行业 + 差公司 | 最差 |

## 二、波特五力模型

### 五力概述

```
           供应商议价能力
                ↓
潜在进入者 → 行业内竞争 ← 替代品威胁
   威胁          ↑
           买方议价能力
```

### 1. 现有竞争者的竞争强度

| 因素 | 竞争激烈 | 竞争温和 |
|------|----------|----------|
| 行业集中度 | 分散 | 集中 |
| 行业增速 | 低/负增长 | 高增长 |
| 固定成本 | 高 | 低 |
| 产品差异化 | 低 | 高 |
| 退出壁垒 | 高 | 低 |

### 2. 潜在进入者的威胁

**进入壁垒**：
| 壁垒类型 | 描述 |
|----------|------|
| 规模经济 | 大规模才能降低成本 |
| 资本需求 | 高资本门槛 |
| 品牌忠诚 | 客户粘性 |
| 渠道控制 | 分销网络 |
| 技术专利 | 知识产权保护 |
| 政策许可 | 牌照、资质 |

### 3. 替代品的威胁

**评估维度**：
- 替代品的相对价格性能
- 消费者转换成本
- 消费者转换意愿

**案例**：
- 数码相机替代胶卷相机
- 流媒体替代DVD
- 电动车替代燃油车

### 4. 供应商的议价能力

| 供应商强 | 供应商弱 |
|----------|----------|
| 供应商集中 | 供应商分散 |
| 无替代投入 | 有替代投入 |
| 行业非主要客户 | 行业是主要客户 |
| 供应商可前向整合 | 无前向整合能力 |

### 5. 买方的议价能力

| 买方强 | 买方弱 |
|--------|--------|
| 买方集中 | 买方分散 |
| 采购量大 | 采购量小 |
| 标准化产品 | 差异化产品 |
| 转换成本低 | 转换成本高 |
| 可后向整合 | 无后向整合能力 |

## 三、产业生命周期

### 四个阶段

```
销量
  |
  |              成熟期
  |            ________
  |           /        \
  |    成长期/          \衰退期
  |        /             \
  |       /               \
  |______/                 \______
  |导入期
  +--------------------------------→ 时间
```

### 各阶段特征

| 阶段 | 增长 | 竞争 | 利润 | 投资重点 |
|------|------|------|------|----------|
| 导入期 | 低 | 少 | 亏损 | 研发、教育市场 |
| 成长期 | 高 | 增加 | 快速增长 | 产能、渠道 |
| 成熟期 | 稳定 | 激烈 | 稳定 | 效率、成本 |
| 衰退期 | 下降 | 减少 | 下降 | 收割、退出 |

### 投资策略

| 阶段 | 策略 |
|------|------|
| 导入期 | 高风险高收益，VC思维 |
| 成长期 | 最佳投资期，成长股策略 |
| 成熟期 | 价值投资，关注分红 |
| 衰退期 | 谨慎，关注转型或清算价值 |

## 四、产业链分析

### 产业链结构

```
上游（原材料/零部件）
        ↓
中游（制造/加工）
        ↓
下游（品牌/渠道/终端）
        ↓
    消费者
```

### 各环节特点

| 环节 | 特点 | 代表 |
|------|------|------|
| 上游 | 资源属性、周期性强 | 矿产、原材料 |
| 中游 | 规模效应、技术壁垒 | 制造业 |
| 下游 | 品牌溢价、贴近市场 | 消费品牌 |

### 利润分配

**微笑曲线**：
```
利润率
  |
  |  研发设计        品牌营销
  |    \              /
  |     \            /
  |      \__制造__/
  |
  +--------------------------------→ 价值链
```

启示：向微笑曲线两端移动

## 五、竞争格局分析

### 行业集中度（CR）

```
CR4 = 前4家市场份额之和
CR8 = 前8家市场份额之和
```

| CR4 | 竞争程度 |
|-----|----------|
| <30% | 分散竞争 |
| 30%-50% | 低集中寡头 |
| 50%-80% | 高集中寡头 |
| >80% | 极高集中 |

### 赫芬达尔指数（HHI）

```
HHI = Σ(各公司市场份额²)
```

| HHI | 竞争程度 |
|-----|----------|
| <1000 | 竞争性市场 |
| 1000-1800 | 适度集中 |
| >1800 | 高度集中 |

### 竞争格局演变

```
完全竞争 → 垄断竞争 → 寡头垄断 → 完全垄断
```

**驱动因素**：
- 规模经济
- 并购整合
- 技术变革
- 政策干预

## 六、行业景气分析

### 领先指标

| 行业 | 领先指标 |
|------|----------|
| 房地产 | 土地成交、新开工 |
| 汽车 | 汽车产销、库存 |
| 消费 | 社零增速、消费者信心 |
| 科技 | 订单、资本开支 |
| 周期品 | 价格、库存、开工率 |

### 景气跟踪框架

| 维度 | 指标 |
|------|------|
| 需求 | 订单、收入、销量 |
| 供给 | 产能、库存、开工率 |
| 价格 | 产品价格、毛利率 |
| 政策 | 行业政策、补贴 |

### 景气周期

```
复苏期：需求回暖、价格企稳
扩张期：需求旺盛、价格上涨、利润扩张
顶部期：需求放缓、产能扩张、竞争加剧
衰退期：需求下滑、价格下跌、利润收缩
底部期：产能出清、库存去化
```

## 八、政策分析

### 政策类型

| 类型 | 影响 |
|------|------|
| 产业政策 | 扶持/限制特定行业 |
| 财政政策 | 税收、补贴 |
| 货币政策 | 融资环境 |
| 监管政策 | 准入、合规要求 |
| 环保政策 | 成本、产能限制 |

### 政策分析框架

1. **政策方向**：支持还是限制？
2. **政策力度**：强推还是温和？
3. **政策持续性**：短期还是长期？
4. **政策执行**：是否严格落地？
5. **边际变化**：是否有新变化？

### 中国特色产业政策

- **"十四五"规划**：五年发展方向
- **双碳目标**：新能源、环保
- **共同富裕**：消费、分配
- **自主可控**：半导体、软件
- **数字经济**：云计算、AI

## 九、技术趋势分析

### 技术采用曲线

```
采用率
  |                    ___滞后者
  |                 __/
  |             ___/    晚期大众
  |         ___/
  |      __/       早期大众
  |   __/
  |__/    早期采用者
  |  创新者
  +--------------------------------→ 时间
```

### Gartner技术成熟度曲线

```
期望
  |
  |      峰值（期望膨胀）
  |      /\
  |     /  \
  |    /    \   复苏期
  |   /      \__/   ___生产成熟
  |  /         \___/
  | /    谷底（幻灭）
  |/ 萌芽期
  +--------------------------------→ 时间
```

### 颠覆性技术识别

**特征**：
- 初期性能不如主流
- 但在某些维度更好
- 快速改进
- 最终颠覆主流

**案例**：
- 智能手机颠覆功能机
- SSD颠覆HDD
- 云计算颠覆传统IT

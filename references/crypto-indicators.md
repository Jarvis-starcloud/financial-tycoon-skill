## 一、长期估值指标

### 1. AHR999 囤币指标

**创建者**：微博用户 ahr999（九神）

**公式**：
```
AHR999 = (BTC价格 / 200日定投成本) × (BTC价格 / 指数增长估值)
```

**关键阈值**：
| 区间 | 含义 | 操作建议 |
|------|------|----------|
| < 0.45 | 🟢 抄底区 | 历史最佳买入时机 |
| 0.45 ~ 1.2 | 🟡 定投区 | 适合定期定额买入 |
| > 1.2 | 🔴 等待/卖出区 | 持币观望或考虑卖出 |

**适用**：长期囤币者（HODLer）

**数据源**：coinglass.com/zh/pro/i/AHR999

---

### 2. MVRV Z-Score

**公式**：
```
Z-Score = (市值 - 实现市值) / 市值标准差
```

**实现市值**：所有BTC按最后一次链上移动时价格计算的总价值（全网持仓成本）

**关键阈值**：
| Z值 | 含义 |
|-----|------|
| > 7 | 🔴 严重高估/顶部信号 |
| 2 ~ 4 | 🟡 中性区间 |
| < 0 | 🟢 严重低估/底部信号 |

**历史峰值**：
- 2017年：接近10
- 2021年：超过7

**注意**：呈边际递减趋势，每轮牛市峰值在降低

**数据源**：glassnode.com, lookintobitcoin.com

### 3. 比特币彩虹图 (Rainbow Chart)

**创建者**：trolololo (2014)

**理论基础**：对数回归模型 + 梅特卡夫定律 + 技术扩散曲线

**九条色带**（自下而上）：

| 色带 | 名称 | 含义 |
|------|------|------|
| 深蓝 | 明显买入区 | 极端折价 |
| 浅蓝 | 积累区 | 战略买家入场 |
| 绿色 | 吸筹区 | 价格有吸引力 |
| 浅绿 | HODL区 | 公允价值，适合长持 |
| 黄色 | 平衡区 | 核心基线 |
| 浅橙 | 买方主导 | FOMO升温 |
| 深橙 | FOMO区 | 泡沫化 |
| 红色 | 超买区 | 考虑获利了结 |
| 深红 | 极端FOMO区 | 泡沫高位 |

**操作原则**：暖色卖、冷色买、黄色持有

**数据源**：blockchaincenter.net/bitcoin-rainbow-chart

### 4. S2F 模型 (Stock-to-Flow)

**创建者**：PlanB

**公式**：
```
S2F比率 = 存量（已发行总量）/ 流量（年新增量）
```

**核心逻辑**：稀缺性驱动价值

**历史S2F比率**：
| 时期 | S2F比率 |
|------|---------|
| 2012减半后 | ~24 |
| 2016减半后 | ~50 |
| 2020减半后 | ~100 |
| 2024减半后 | ~200 |

**局限性**：
- 只考虑供应端稀缺性
- 忽略需求端变化
- 近几年偏差增大

**数据源**：stats.buybitcoinworldwide.com

### 5. Pi Cycle Top Indicator

**原理**：
```
111日移动均线(111DMA) vs 350日移动均线×2(350DMA×2)
```

**顶部信号**：111DMA 向上穿越 350DMA×2

**历史验证**：
- 2017年顶部前出现交叉
- 2021年顶部前出现交叉

**用途**：极端过热时刻的视觉预警工具

**数据源**：lookintobitcoin.com

## 二、链上活动指标

### 6. NUPL (Net Unrealized Profit/Loss)

**公式**：
```
NUPL = (市值 - 实现市值) / 市值
```

**关键阈值**：
| NUPL | 情绪 | 含义 |
|------|------|------|
| > 0.75 | 贪婪 | 🔴 顶部区域 |
| 0.5 ~ 0.75 | 乐观/信念 | 🟡 牛市中段 |
| 0 ~ 0.5 | 希望/焦虑 | 🟡 震荡区 |
| < 0 | 恐惧/投降 | 🟢 底部区域 |

**历史峰值**：
- 2017年：超过0.8
- 2021年：超过0.7

**含义**：衡量全网未实现盈亏，高值=人人浮盈，抛售风险大

**数据源**：glassnode.com

---

### 7. SOPR (Spent Output Profit Ratio)

**公式**：
```
SOPR = 链上转移币的卖出价值 / 买入价值
```

**关键阈值**：
| SOPR | 含义 |
|------|------|
| > 1 | 市场整体盈利出售（牛市确认） |
| = 1 | 盈亏平衡（关键支撑/阻力） |
| < 1 | 市场整体亏损出售（熊市/投降） |

**变体**：
- **STH-SOPR**：短期持有者SOPR，更灵敏的短线指标
- **LTH-SOPR**：长期持有者SOPR

**用途**：判断链上卖压和市场情绪转折点

**数据源**：glassnode.com

---

### 8. Puell Multiple (普尔倍数)

**公式**：
```
Puell Multiple = 矿工每日收入（USD）/ 365日移动平均值
```

**关键阈值**：
| 倍数 | 含义 |
|------|------|
| > 4 | 🔴 矿工暴利，顶部信号 |
| 1 ~ 2 | 🟡 正常区间 |
| < 0.5 | 🟢 矿工投降，底部信号 |

**历史峰值**：
- 2017年：超过7
- 2021年：超过3

**含义**：矿工收入过高=卖压积累，过低=抛售枯竭

**数据源**：glassnode.com, lookintobitcoin.com

---

### 9. NVT 比率 (Network Value to Transactions)

**公式**：
```
NVT = 网络市值 / 链上日交易量（USD）
```

**类比**：加密货币界的"市盈率（P/E）"

**判断**：
| NVT | 含义 |
|-----|------|
| 过高 | 估值相对交易量偏贵（泡沫风险） |
| 过低 | 估值相对交易量偏低（低估机会） |

**用途**：从"网络效用"角度评估估值合理性

**数据源**：glassnode.com

## 三、技术与情绪指标

### 10. 梅耶倍数 (Mayer Multiple)

**创建者**：Trace Mayer

**公式**：
```
Mayer Multiple = BTC当前价格 / 200日移动均线(MA200)
```

**关键阈值**：
| 倍数 | 含义 |
|------|------|
| > 2.4 | 🔴 超买，价格远高于长期均值 |
| ~1.0 | 🟡 均值回归区 |
| < 0.8 | 🟢 超卖，价格远低于长期均值 |

**用途**：衡量价格偏离长期趋势的程度

**数据源**：mayermultiple.info

---

### 11. 恐惧与贪婪指数 (Fear & Greed Index)

**数据源**：Alternative.me / CoinGlass

**范围**：0 ~ 100

**区间含义**：
| 区间 | 情绪 | 操作建议 |
|------|------|----------|
| 0 ~ 24 | 极度恐惧 | 🟢 可能是买入机会 |
| 25 ~ 49 | 恐惧 | 🟡 偏谨慎 |
| 50 | 中性 | 🟡 观望 |
| 51 ~ 74 | 贪婪 | 🟡 偏乐观 |
| 75 ~ 100 | 极度贪婪 | 🔴 可能是卖出信号 |

**构成因素**：
- 价格波动
- 交易量
- 社交媒体
- 市场调查
- BTC占比
- Google趋势

**数据源**：alternative.me/crypto/fear-and-greed-index

---

### 12. 资金费率 (Funding Rate)

**来源**：永续合约交易所

**含义**：多空双方互相支付的费用

**判断**：
| 费率 | 含义 |
|------|------|
| 持续正费率 | 多头拥挤，可能回调 |
| 持续负费率 | 空头拥挤，可能反弹 |
| 极端费率 | 反转信号 |

**数据源**：coinglass.com

---

### 13. 多空比 (Long/Short Ratio)

**来源**：各大交易所

**用途**：判断主力方向

**类型**：
- 大户持仓多空比
- 散户持仓多空比
- 账户数多空比

**数据源**：coinglass.com, binance

---

### 14. 清算数据 (Liquidations)

**含义**：杠杆仓位被强制平仓

**判断**：
| 现象 | 含义 |
|------|------|
| 大规模多头清算 | 可能见底 |
| 大规模空头清算 | 可能见顶 |
| 清算高峰 | 市场极端点 |

**数据源**：coinglass.com, coinalyze.net

## 四、资金流向指标

### 15. 交易所净流入/流出

**判断**：
| 流向 | 含义 |
|------|------|
| 净流入 | BTC进入交易所，卖压增加 |
| 净流出 | BTC离开交易所，囤币增加（看涨） |

**数据源**：glassnode.com, cryptoquant.com

---

### 16. 稳定币流入

**判断**：
- 稳定币大量流入交易所 → 买盘准备
- 关注USDT/USDC市值变化

**数据源**：glassnode.com, defillama.com

---

### 17. 巨鲸动向

**关注**：
- 大额钱包增持/减持
- 前100地址变化
- 休眠地址活动

**数据源**：bitinfocharts.com, whale-alert.io

---

### 18. ETF资金流

**关注**：
- 现货ETF流入/流出
- GBTC、IBIT等产品
- 机构资金风向标

**数据源**：各ETF发行商、coinglass.com

## 七、重要提醒

1. **指标滞后**：链上数据有延迟
2. **不是绝对**：任何指标都会失效
3. **结合使用**：多指标交叉验证
4. **宏观优先**：先看美联储、美元
5. **风险第一**：保护本金最重要

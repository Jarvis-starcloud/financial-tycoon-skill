## 一、行为金融学概述

### 与传统金融的区别

| 传统金融 | 行为金融 |
|----------|----------|
| 理性人假设 | 有限理性 |
| 市场有效 | 市场可能非有效 |
| 风险=波动 | 风险感知因人而异 |
| 最优决策 | 满意即可 |

### 核心观点

1. 投资者不完全理性
2. 认知偏误系统性存在
3. 市场价格可能偏离价值
4. 情绪影响市场波动

## 二、认知偏误

### 信息处理偏误

| 偏误 | 表现 | 后果 |
|------|------|------|
| **确认偏误** | 只关注支持自己的信息 | 忽视风险信号 |
| **锚定效应** | 被初始信息过度影响 | 调整不充分 |
| **可得性偏误** | 高估容易想到的事件 | 决策偏差 |
| **代表性偏误** | 小样本推断总体 | 过度外推 |
| **后见之明** | "我早就知道" | 高估预测能力 |

### 情绪偏误

| 偏误 | 表现 | 后果 |
|------|------|------|
| **损失厌恶** | 损失痛苦>收益快乐（约2:1） | 不愿止损 |
| **后悔厌恶** | 害怕做出错误决定 | 不敢行动 |
| **过度自信** | 高估自己的能力 | 交易过频 |
| **乐观偏误** | 低估负面事件概率 | 风险意识不足 |
| **禀赋效应** | 高估已拥有的东西 | 不愿卖出 |

### 自我控制偏误

| 偏误 | 表现 | 后果 |
|------|------|------|
| **现时偏好** | 即时满足>延迟满足 | 储蓄不足 |
| **自制力不足** | 明知故犯 | 无法执行计划 |
| **心理账户** | 不同来源的钱区别对待 | 非理性配置 |

## 三、前景理论

### 卡尼曼和特沃斯基的发现

**核心要点**：

1. **参考点效应**：收益/损失是相对参考点而言
2. **损失厌恶**：损失的痛苦约为等额收益快乐的2倍
3. **敏感性递减**：边际效用递减（无论盈亏）
4. **概率权重**：高估小概率事件，低估大概率事件

### 价值函数

```
        收益
          |      /
          |    /
          |  /
          |/
----------|----------
         /|
       /  |
     /    |
   /      |
  损失（更陡峭）
```

### 投资影响

| 现象 | 前景理论解释 |
|------|--------------|
| 不愿止损 | 损失厌恶，继续持有期望回本 |
| 过早止盈 | 收益区域风险厌恶 |
| 追买彩票股 | 高估小概率大收益 |
| 低估保险价值 | 低估小概率大损失 |

## 四、市场异象

### 动量效应

**定义**：过去表现好的股票未来继续表现好

**可能解释**：
- 投资者反应不足
- 正反馈交易
- 信息逐步扩散

**应用**：动量策略、趋势跟踪

### 反转效应

**定义**：长期（3-5年）过去表现差的股票未来表现好

**可能解释**：
- 过度反应后修正
- 均值回归

### 规模效应

**定义**：小市值股票长期跑赢大市值

**可能解释**：
- 风险溢价
- 信息不对称
- 流动性溢价

### 价值效应

**定义**：低估值股票长期跑赢高估值

**可能解释**：
- 风险溢价
- 过度外推成长
- 均值回归

### 日历效应

| 效应 | 现象 |
|------|------|
| 一月效应 | 一月份收益较高 |
| 周一效应 | 周一收益较低 |
| 节假日效应 | 节前收益较高 |
| 月末效应 | 月末收益较高 |

## 五、群体行为

### 羊群效应

**定义**：投资者模仿他人决策

**原因**：
1. 信息不对称
2. 声誉考量（机构）
3. 从众心理
4. 正反馈机制

**后果**：
- 放大市场波动
- 形成泡沫和崩盘
- 价格偏离价值

### 信息瀑布

**定义**：后来者忽略自己信息，跟随先行者

**形成过程**：
1. 先行者根据私有信息行动
2. 后来者观察先行者行为
3. 放弃自己信息，跟随
4. 形成瀑布效应

### 社交媒体放大

- 信息传播加速
- 情绪传染更快
- FOMO（错失恐惧）
- 网红效应

## 六、情绪与市场

### 投资者情绪指标

| 指标 | 来源 | 含义 |
|------|------|------|
| 恐惧贪婪指数 | CNN | 综合情绪 |
| VIX | 期权市场 | 恐慌程度 |
| 融资余额 | 交易所 | 杠杆情绪 |
| 新开户数 | 券商 | 入场热情 |
| 基金申赎 | 基金公司 | 散户行为 |
| 舆情热度 | 社交媒体 | 关注度 |

### 情绪周期

```
牛市初期：怀疑 → 乐观
牛市中期：乐观 → 兴奋
牛市末期：兴奋 → 狂热（FOMO）
熊市初期：否认 → 恐惧
熊市中期：恐惧 → 绝望
熊市末期：绝望 → 麻木 → 怀疑（重新开始）
```

### 逆向投资

**核心理念**：在别人恐惧时贪婪，在别人贪婪时恐惧

**实践要点**：
1. 识别极端情绪
2. 有独立判断能力
3. 能承受短期不适
4. 有足够耐心

## 七、行为金融应用

### 克服偏误的方法

| 偏误 | 应对策略 |
|------|----------|
| 确认偏误 | 主动寻找反面证据 |
| 损失厌恶 | 机械化止损规则 |
| 过度自信 | 记录预测、事后复盘 |
| 锚定效应 | 关注当前价值非成本 |
| 从众心理 | 独立研究、逆向思考 |
| 心理账户 | 整体资产视角 |

### 投资决策清单

1. **买入前**：
   - 投资逻辑是什么？
   - 什么会证明我错了？
   - 我愿意亏多少？

2. **持有中**：
   - 逻辑还成立吗？
   - 是否需要调整仓位？
   - 是否被情绪影响？

3. **卖出后**：
   - 对还是错？为什么？
   - 有什么教训？
   - 如何改进？

### 纪律投资

- **定期定额**：克服择时恐惧
- **机械化规则**：减少情绪干扰
- **资产配置**：避免集中押注
- **再平衡**：强制低买高卖

## 八、市场泡沫

### 泡沫特征

| 阶段 | 特征 |
|------|------|
| 隐蔽期 | 少数人发现机会 |
| 觉醒期 | 更多人关注，价格上涨 |
| 狂热期 | 大众参与，媒体热炒 |
| 衰落期 | 聪明钱退出 |
| 崩溃期 | 恐慌性抛售 |

### 识别信号

- [ ] 价格脱离基本面
- [ ] 交易量异常放大
- [ ] 杠杆水平飙升
- [ ] 新参与者大量涌入
- [ ] "这次不一样"论调
- [ ] 媒体狂热报道
- [ ] 出租车司机都在讨论

### 历史案例

| 泡沫 | 时间 | 标的 |
|------|------|------|
| 郁金香泡沫 | 1637 | 郁金香球茎 |
| 南海泡沫 | 1720 | 南海公司股票 |
| 大萧条 | 1929 | 美股 |
| 日本泡沫 | 1989 | 日本股市房市 |
| 科网泡沫 | 2000 | 互联网股票 |

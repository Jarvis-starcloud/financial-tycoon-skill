## 一、三大财务报表

### 资产负债表（Balance Sheet）

**结构**：资产 = 负债 + 所有者权益

| 资产（左/上） | 负债+权益（右/下） |
|--------------|-------------------|
| 流动资产 | 流动负债 |
| 非流动资产 | 非流动负债 |
| | 所有者权益 |

**关键项目**：
| 项目 | 关注点 |
|------|--------|
| 货币资金 | 现金储备是否充足 |
| 应收账款 | 回款能力、坏账风险 |
| 存货 | 周转率、减值风险 |
| 固定资产 | 折旧政策、产能利用 |
| 商誉 | 收购溢价、减值风险 |
| 短期借款 | 短期偿债压力 |
| 应付账款 | 对供应商议价能力 |
| 长期借款 | 长期财务杠杆 |

### 利润表（Income Statement）

**结构**：收入 - 成本费用 = 利润

```
营业收入
 - 营业成本
 = 毛利润
 - 三费（销售/管理/财务）
 - 研发费用
 + 其他收益
 = 营业利润
 +/- 营业外收支
 = 利润总额
 - 所得税
 = 净利润
```

**关键指标**：
| 指标 | 公式 | 意义 |
|------|------|------|
| 毛利率 | 毛利/收入 | 产品竞争力 |
| 净利率 | 净利/收入 | 整体盈利能力 |
| 费用率 | 三费/收入 | 成本控制 |

### 现金流量表（Cash Flow Statement）

**三大类现金流**：

| 类别 | 内容 | 健康状态 |
|------|------|----------|
| 经营活动 | 主营业务产生 | 应为正 |
| 投资活动 | 资产购置/处置 | 扩张期为负 |
| 筹资活动 | 借款/还款/分红 | 视情况 |

**关键关系**：
- 经营现金流 > 净利润：利润质量高
- 经营现金流 < 净利润：可能存在问题

## 二、核心财务比率

### 盈利能力

| 指标 | 公式 | 参考值 |
|------|------|--------|
| 毛利率 | 毛利/收入 | 行业差异大 |
| 净利率 | 净利/收入 | >10%较好 |
| ROE | 净利/净资产 | >15%优秀 |
| ROA | 净利/总资产 | >5%较好 |
| ROIC | 税后营业利润/投入资本 | >WACC |

### 营运能力

| 指标 | 公式 | 意义 |
|------|------|------|
| 应收账款周转天数 | 365/(收入/平均应收) | 越短越好 |
| 存货周转天数 | 365/(成本/平均存货) | 越短越好 |
| 应付账款周转天数 | 365/(成本/平均应付) | 越长越好 |
| 现金周期 | 应收天数+存货天数-应付天数 | 越短越好 |

### 偿债能力

| 指标 | 公式 | 警戒线 |
|------|------|--------|
| 流动比率 | 流动资产/流动负债 | >1 |
| 速动比率 | (流动资产-存货)/流动负债 | >0.7 |
| 资产负债率 | 负债/资产 | <70% |
| 利息保障倍数 | EBIT/利息支出 | >3 |

### 成长能力

| 指标 | 公式 |
|------|------|
| 营收增长率 | (本期收入-上期)/上期 |
| 净利润增长率 | (本期净利-上期)/上期 |
| 扣非净利润增长率 | 剔除非经常性损益 |

## 三、杜邦分析

### 核心公式

```
ROE = 净利率 × 资产周转率 × 权益乘数
    = (净利/收入) × (收入/资产) × (资产/净资产)
```

### 三条路径提升ROE

| 路径 | 代表行业 | 特点 |
|------|----------|------|
| 高净利率 | 奢侈品、软件 | 差异化 |
| 高周转 | 零售、快消 | 规模效应 |
| 高杠杆 | 金融、地产 | 风险放大 |

### 杜邦分析应用

1. 纵向对比：同公司不同年份
2. 横向对比：同行业不同公司
3. 识别驱动因素变化

## 四、财报质量分析

### 利润质量

**警惕信号**：
| 现象 | 可能问题 |
|------|----------|
| 收入增长但现金流为负 | 虚增收入 |
| 应收账款增速>收入增速 | 放宽信用 |
| 存货积压 | 销售不畅 |
| 非经常性损益占比高 | 主业不佳 |
| 关联交易占比高 | 利益输送 |

### 资产质量

**减值风险项目**：
- 商誉（收购溢价）
- 长期股权投资
- 固定资产
- 存货
- 应收账款

### 现金流质量

**健康状态**：
```
经营现金流为正
 + 投资适度（负）
 + 筹资灵活
 = 自由现金流为正
```

## 五、财务舞弊识别

### 常见手法

| 手法 | 特征 |
|------|------|
| 虚增收入 | 关联交易、期末突击 |
| 少计费用 | 资本化、跨期调节 |
| 虚构资产 | 存货异常、在建工程 |
| 隐藏负债 | 表外融资、担保 |
| 操纵现金流 | 票据融资、关联方 |

### 预警信号

1. 与行业趋势背离
2. 频繁更换审计师
3. 管理层频繁减持
4. 会计政策频繁变更
5. 复杂难懂的交易结构

### 分析技巧

**贝尼什M-Score模型**：
- DSRI（应收账款指数）
- GMI（毛利率指数）
- AQI（资产质量指数）
- SGI（销售增长指数）
- DEPI（折旧指数）

M-Score > -2.22：舞弊概率高

## 六、不同行业财报特点

### 制造业

- 重点：产能利用率、存货周转、固定资产
- 关注：折旧政策、资本开支

### 零售业

- 重点：同店增长、坪效、存货周转
- 关注：租金成本、现金周期

### 科技业

- 重点：研发投入、无形资产、用户数据
- 关注：收入确认政策、递延收入

### 金融业

- 重点：不良率、拨备覆盖率、净息差
- 关注：表外资产、资本充足率

### 地产业

- 重点：土储、预售、回款
- 关注：合同负债、三道红线

## 七、季报/年报阅读顺序

### 快速阅读法

1. **审计意见**：标准/非标
2. **财务摘要**：收入、利润、现金流
3. **管理层讨论**：战略、展望
4. **分部信息**：业务结构变化
5. **重大事项**：诉讼、担保、承诺
6. **详细报表**：深入分析

### 关键页面

| 内容 | 作用 |
|------|------|
| 审计报告 | 可靠性背书 |
| 董事会报告 | 定性描述 |
| 财务报告 | 定量数据 |
| 附注 | 会计政策、明细 |
| 内部控制评价 | 治理水平 |

## 九、财务预测方法

### 收入预测

1. **自上而下**：行业规模 × 市占率
2. **自下而上**：产品销量 × 单价
3. **历史趋势**：增长率外推

### 成本费用预测

- 变动成本：与收入挂钩
- 固定成本：相对稳定
- 半变动成本：基数+变动

### 资本开支预测

- 维护性资本开支
- 扩张性资本开支
- 折旧年限匹配

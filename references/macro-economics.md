## 一、核心概念

### GDP (国内生产总值)

**定义**：一定时期内一国境内生产的所有最终商品和服务的市场价值总和。

**三种核算方法**：
1. **生产法**：总产出 - 中间投入
2. **收入法**：劳动报酬 + 生产税净额 + 折旧 + 营业盈余
3. **支出法**：C + I + G + (X - M)

**关键指标**：
- **名义GDP**：按当期价格计算
- **实际GDP**：剔除通胀后（可比价格）
- **GDP增速**：同比/环比变化率
- **人均GDP**：GDP/人口

**中国GDP相关**：
- 目标增速：通常设定在5%~6%区间
- 统计局季度公布
- 关注"三驾马车"贡献度：消费、投资、出口

## 二、通货膨胀指标

### CPI (居民消费价格指数)

**构成权重（中国）**：
| 分类 | 权重 |
|------|------|
| 食品烟酒 | ~30% |
| 居住 | ~20% |
| 交通通信 | ~13% |
| 教育文娱 | ~10% |
| 医疗保健 | ~8% |
| 衣着 | ~6% |
| 生活用品 | ~5% |
| 其他 | ~8% |

**关键阈值**：
- < 0%：通缩（需警惕）
- 0% ~ 2%：低通胀（可能需要刺激）
- 2% ~ 3%：温和通胀（央行目标区间）
- 3% ~ 5%：中等通胀（需关注）
- > 5%：高通胀（需紧缩）

### PPI (生产者价格指数)

- 反映工业品出厂价格变化
- 通常领先CPI 2-3个月
- PPI → CPI传导链条

### PCE (个人消费支出物价指数)

- 美联储首选通胀指标
- 与CPI区别：权重动态调整、覆盖范围更广
- 核心PCE：剔除食品和能源

## 三、就业指标

### 非农就业 (Non-Farm Payrolls)

**美国劳工部月度公布**：
- 新增非农就业人数
- 失业率
- 劳动参与率
- 平均时薪

**关键解读**：
| 指标 | 强劲 | 疲软 |
|------|------|------|
| 新增就业 | >20万 | <10万 |
| 失业率 | <4% | >5% |
| 时薪增速 | >3% | <2% |

### 中国就业指标

- 城镇调查失业率（目标<5.5%）
- 新增城镇就业（目标1100万+）
- PMI就业分项

## 四、PMI (采购经理指数)

### 构成分项

| 分项 | 权重 |
|------|------|
| 新订单 | 30% |
| 生产 | 25% |
| 就业 | 20% |
| 供应商配送 | 15% |
| 库存 | 10% |

### 解读标准

- **50%荣枯线**：>50扩张，<50收缩
- **制造业PMI**：工业景气
- **非制造业PMI**：服务业景气
- **综合PMI**：整体经济

### 重要PMI发布

| 国家/地区 | 发布机构 |
|-----------|----------|
| 美国 | ISM |
| 欧元区 | Markit |
| 中国官方 | 统计局 |
| 中国财新 | Markit |

## 五、利率与货币政策

### 美联储政策工具

| 工具 | 作用 |
|------|------|
| 联邦基金利率 | 基准利率 |
| 贴现率 | 紧急贷款利率 |
| 准备金利率 | 超额准备金利息 |
| QE/QT | 资产购买/缩表 |

### 利率传导机制

```
联邦基金利率 → 国债收益率 → 企业债利率 → 贷款利率 → 实体经济
```

### 中国利率体系

| 利率 | 性质 |
|------|------|
| 存款基准利率 | 历史遗留 |
| LPR | 贷款市场报价利率 |
| MLF利率 | 中期借贷便利 |
| OMO利率 | 公开市场操作 |

### 利率政策周期

```
降息周期 → 经济复苏 → 加息周期 → 经济收缩 → 降息周期
```

## 六、经济周期

### 基钦周期（短周期）
- 时长：3-4年
- 驱动：库存变化

### 朱格拉周期（中周期）
- 时长：7-11年
- 驱动：设备投资

### 康德拉季耶夫周期（长周期）
- 时长：50-60年
- 驱动：技术革命

### 美林时钟

| 阶段 | 经济 | 通胀 | 配置 |
|------|------|------|------|
| 复苏 | ↑ | ↓ | 股票 |
| 过热 | ↑ | ↑ | 大宗商品 |
| 滞涨 | ↓ | ↑ | 现金 |
| 衰退 | ↓ | ↓ | 债券 |

## 八、财政政策

### 财政工具

| 工具 | 扩张性 | 收缩性 |
|------|--------|--------|
| 税收 | 减税 | 加税 |
| 支出 | 增支 | 减支 |
| 转移支付 | 增加 | 减少 |

### 财政乘数效应

- 政府支出乘数 > 税收乘数
- 边际消费倾向越高，乘数越大

### 财政可持续性

- 赤字率（赤字/GDP，警戒线3%）
- 负债率（债务/GDP，警戒线60%）
- 利息支出/财政收入

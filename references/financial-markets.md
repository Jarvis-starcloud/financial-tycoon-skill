## 一、金融市场分类

### 按交易对象

| 市场 | 交易标的 | 特点 |
|------|----------|------|
| 货币市场 | 短期债务工具（<1年） | 流动性高、风险低 |
| 资本市场 | 股票、长期债券 | 期限长、收益高 |
| 外汇市场 | 货币兑换 | 24小时、杠杆高 |
| 衍生品市场 | 期货、期权、互换 | 杠杆、对冲 |
| 大宗商品市场 | 原油、黄金、农产品 | 实物交割/金融属性 |

### 按发行与流通

- **一级市场**：新证券发行（IPO、增发）
- **二级市场**：已发行证券交易

### 按交易场所

- **场内市场**：交易所（标准化、集中）
- **场外市场**：OTC（定制化、分散）

## 三、债券市场

### 债券类型

| 类型 | 发行人 | 特点 |
|------|--------|------|
| 国债 | 政府 | 无风险基准 |
| 地方债 | 地方政府 | 一般/专项 |
| 企业债 | 企业 | 信用风险 |
| 公司债 | 上市公司 | 交易所交易 |
| 金融债 | 金融机构 | 政策性/商业性 |
| 可转债 | 上市公司 | 转股权 |

### 债券要素

- **面值**：通常100元
- **票面利率**：固定/浮动
- **期限**：到期日
- **信用评级**：AAA/AA+/AA...
- **付息方式**：按年/半年/到期一次

### 债券定价

**到期收益率（YTM）公式**：
```
价格 = Σ(票息/(1+YTM)^t) + 面值/(1+YTM)^n
```

**债券价格与利率反向关系**：
- 利率上升 → 债券价格下跌
- 利率下降 → 债券价格上涨

### 久期与凸性

- **久期**：价格对利率变化的敏感度
- **凸性**：久期的变化率
- 久期越长，利率风险越大

## 五、衍生品市场

### 期货

**特点**：
- 标准化合约
- 交易所交易
- 保证金交易（杠杆）
- 每日无负债结算
- 到期交割或平仓

**主要品种**：
- 商品期货：原油、黄金、铜、农产品
- 金融期货：股指、国债、外汇

### 期权

**期权类型**：
| 维度 | 分类 |
|------|------|
| 权利 | 看涨（Call）/看跌（Put） |
| 行权 | 欧式/美式/百慕大 |
| 价值 | 实值/平值/虚值 |

**希腊字母**：
- **Delta**：标的价格敏感度
- **Gamma**：Delta变化率
- **Theta**：时间价值衰减
- **Vega**：波动率敏感度

### 互换（Swap）

- **利率互换**：固定↔浮动
- **货币互换**：不同货币现金流
- **信用违约互换（CDS）**：信用风险转移

## 六、大宗商品市场

### 主要品种

| 类别 | 品种 |
|------|------|
| 能源 | 原油、天然气、煤炭 |
| 贵金属 | 黄金、白银、铂金 |
| 工业金属 | 铜、铝、锌、镍 |
| 农产品 | 大豆、玉米、小麦、棉花 |

### 原油市场

**主要基准**：
- **WTI**：美国西德克萨斯（纽约）
- **Brent**：北海布伦特（伦敦）
- **Dubai**：中东迪拜

**影响因素**：
1. OPEC+产量决策
2. 地缘政治（中东、俄罗斯）
3. 全球经济增长
4. 美元强弱
5. 库存数据（EIA周报）

### 黄金市场

**定价机制**：
- 伦敦金（现货基准）
- COMEX（期货）
- 上海金（中国定价权）

**黄金属性**：
- 避险资产（地缘风险）
- 抗通胀资产
- 与美元/实际利率负相关

## 七、市场参与者

### 机构投资者

| 类型 | 特点 |
|------|------|
| 公募基金 | 公开募集、监管严格 |
| 私募基金 | 高净值、策略灵活 |
| 保险资金 | 长期、稳健 |
| 养老金 | 超长期、低风险偏好 |
| 社保基金 | 国家战略投资者 |
| QFII/RQFII | 境外机构 |
| 北向资金 | 沪深港通外资 |

### 个人投资者

- 散户：资金量小、信息劣势
- 大户/游资：资金量大、短线为主
- 高净值客户：私人银行服务

### 做市商

- 提供流动性
- 赚取买卖价差
- 承担库存风险

## 八、市场效率理论

### 有效市场假说（EMH）

| 形式 | 信息反映 | 超额收益可能性 |
|------|----------|----------------|
| 弱式有效 | 历史价格 | 技术分析无效 |
| 半强式有效 | 公开信息 | 基本面分析无效 |
| 强式有效 | 所有信息 | 内幕交易也无效 |

### 市场异象（对EMH挑战）

- 动量效应
- 规模效应
- 价值效应
- 日历效应

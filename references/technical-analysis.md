## 一、技术分析三大假设

1. **市场行为包容消化一切**：价格反映所有信息
2. **价格以趋势方式演变**：趋势一旦形成倾向于延续
3. **历史会重演**：人性不变，图形会重复

## 二、K线基础

### K线构成

```
    ┌─ 上影线
    │
■■■■■ ← 收盘价（阳线）/ 开盘价（阴线）
■■■■■
■■■■■ ← 实体
■■■■■
■■■■■ ← 开盘价（阳线）/ 收盘价（阴线）
    │
    └─ 下影线
```

### 基本K线形态

| 形态 | 特征 | 含义 |
|------|------|------|
| 大阳线 | 实体长、无影线 | 强烈看涨 |
| 大阴线 | 实体长、无影线 | 强烈看跌 |
| 十字星 | 开收价相近 | 转折信号 |
| 锤子线 | 下影线长、上影短 | 底部反转 |
| 倒锤子 | 上影线长、下影短 | 底部反转（需确认） |
| 上吊线 | 同锤子但在高位 | 顶部信号 |
| 射击之星 | 上影长、在高位 | 顶部信号 |

### 组合K线形态

**看涨形态**：
- **晨星**：阴线 + 十字星 + 阳线
- **看涨吞没**：小阴 + 大阳包住
- **红三兵**：三根连续阳线

**看跌形态**：
- **黄昏星**：阳线 + 十字星 + 阴线
- **看跌吞没**：小阳 + 大阴包住
- **三只乌鸦**：三根连续阴线

## 三、趋势理论

### 道氏理论

**三种趋势**：
1. **主要趋势**：大势（月/年级别）
2. **次要趋势**：调整（周级别）
3. **短暂趋势**：波动（日内/数日）

**趋势确认**：
- 上升趋势：高点更高、低点更高
- 下降趋势：高点更低、低点更低
- 横盘整理：高低点无明显规律

### 趋势线画法

- **上升趋势线**：连接两个以上低点
- **下降趋势线**：连接两个以上高点
- **有效性**：触及次数越多越有效

### 支撑与阻力

**形成来源**：
1. 前期高低点
2. 整数关口（心理价位）
3. 缺口位置
4. 均线位置
5. 成交密集区

**转换规则**：
- 阻力突破后变支撑
- 支撑跌破后变阻力

## 四、形态分析

### 反转形态

#### 头肩顶/底

```
头肩顶：           头肩底：
    ∧                  
   /  \              ____
  /    \            /    \
 / ∧  ∧ \          /  ∨  \
/左肩 右肩\       /左肩 右肩\
───颈线───       ───颈线───
```

**交易要点**：
- 成交量：头部最大、右肩萎缩
- 确认：突破颈线
- 目标：头部到颈线距离

#### 双顶/双底

```
双顶(M顶)：         双底(W底)：
 ∧    ∧              
/  \  /  \          ____
    \/              /    \
    颈线           /  ∨∨  \
                   ─颈线──
```

#### 三重顶/底

- 类似双顶/底，但有三个高/低点
- 信号更强

#### V型反转

- 快速下跌后急速反弹
- 无明显整理
- 成交量巨大

### 持续形态

#### 三角形

| 类型 | 形态 | 突破方向 |
|------|------|----------|
| 对称三角 | 高低点收敛 | 随原趋势 |
| 上升三角 | 顶平底升 | 向上 |
| 下降三角 | 底平顶降 | 向下 |

#### 旗形与楔形

- **旗形**：平行四边形，反向倾斜
- **楔形**：收敛三角形，反向倾斜
- 都是短期整理形态

#### 矩形整理

- 价格在水平区间内震荡
- 上下边界明确
- 突破方向跟随原趋势

## 五、均线系统

### 常用均线周期

| 周期 | 含义 | 作用 |
|------|------|------|
| MA5 | 周线 | 短期趋势 |
| MA10 | 半月线 | 短期趋势 |
| MA20 | 月线 | 中期趋势 |
| MA60 | 季线 | 中期趋势 |
| MA120 | 半年线 | 长期趋势 |
| MA250 | 年线 | 牛熊分界 |

### 均线排列

- **多头排列**：短期均线在上，长期在下（牛市）
- **空头排列**：长期均线在上，短期在下（熊市）

### 金叉与死叉

- **金叉**：短期均线上穿长期均线（买入信号）
- **死叉**：短期均线下穿长期均线（卖出信号）

### 葛兰碧八大法则

**买入法则**（4条）：
1. 均线由降转平/升，价格从下向上穿越
2. 价格跌破均线后快速反弹
3. 价格在均线上方回落但未跌破
4. 价格远离均线急跌（乖离过大）

**卖出法则**（4条）：同上的镜像

## 六、技术指标

### MACD（指数平滑异同移动平均线）

**构成**：
- **DIF**：12日EMA - 26日EMA
- **DEA**：DIF的9日EMA
- **柱状图**：DIF - DEA

**信号**：
| 信号 | 条件 | 含义 |
|------|------|------|
| 金叉 | DIF上穿DEA | 买入 |
| 死叉 | DIF下穿DEA | 卖出 |
| 零轴上金叉 | 强势买入 | 做多 |
| 零轴下死叉 | 强势卖出 | 做空 |
| 顶背离 | 价格新高MACD不创新高 | 见顶 |
| 底背离 | 价格新低MACD不创新低 | 见底 |

### RSI（相对强弱指标）

**公式**：
```
RSI = 100 - 100/(1+RS)
RS = N日平均涨幅 / N日平均跌幅
```

**参数**：通常14日

**信号**：
| 区间 | 含义 |
|------|------|
| >70 | 超买，可能回调 |
| 50 | 多空分界 |
| <30 | 超卖，可能反弹 |

**背离**：与MACD类似

### KDJ（随机指标）

**构成**：
- **K值**：RSV的3日平滑
- **D值**：K的3日平滑
- **J值**：3K - 2D

**信号**：
| 信号 | 条件 |
|------|------|
| 超买 | K/D>80，J>100 |
| 超卖 | K/D<20，J<0 |
| 金叉 | K上穿D |
| 死叉 | K下穿D |

### 布林带（BOLL）

**构成**：
- **中轨**：20日MA
- **上轨**：中轨 + 2倍标准差
- **下轨**：中轨 - 2倍标准差

**信号**：
| 现象 | 含义 |
|------|------|
| 带宽收窄 | 变盘前兆 |
| 带宽扩大 | 趋势形成 |
| 价格触及上轨 | 强势/超买 |
| 价格触及下轨 | 弱势/超卖 |
| 沿上轨运行 | 强势趋势 |

### 成交量（VOL）

**量价关系**：
| 关系 | 含义 |
|------|------|
| 量价齐升 | 健康上涨 |
| 量价背离 | 趋势可能反转 |
| 放量突破 | 有效突破 |
| 缩量回调 | 健康调整 |
| 地量地价 | 底部信号 |
| 天量天价 | 顶部信号 |

## 七、波浪理论

### 基本结构

```
        5浪
         ∧
    3浪  / \
     ∧  /   \  A浪
    / \/     \  /\
   /  4浪    \/  \
  /           B浪 \
 /                 \ C浪
1浪                 \
```

**推动浪**（5浪）：顺势
**调整浪**（3浪）：逆势

### 浪的特征

| 浪 | 特征 |
|------|------|
| 1浪 | 怀疑中启动 |
| 2浪 | 回调测试（不破1浪起点） |
| 3浪 | 最强最长（往往延伸） |
| 4浪 | 温和调整（不进入1浪区域） |
| 5浪 | 最后冲刺（可能力度不如3浪） |
| A浪 | 下跌开始 |
| B浪 | 反弹（诱多陷阱） |
| C浪 | 主跌浪 |

### 斐波那契回调

常用回调位：
- 23.6%
- 38.2%
- 50%
- 61.8%（黄金分割）
- 78.6%

## 八、缺口分析

### 缺口类型

| 类型 | 位置 | 意义 |
|------|------|------|
| 普通缺口 | 整理区间 | 很快回补 |
| 突破缺口 | 突破关键位 | 不易回补 |
| 持续缺口 | 趋势中段 | 量度目标 |
| 衰竭缺口 | 趋势末端 | 反转信号 |

### 缺口操作原则

- 突破缺口：顺势跟进
- 衰竭缺口：反向操作
- 缺口回补：重要支撑/阻力

## 九、技术分析应用原则

### 多周期共振

```
周线定方向 → 日线找机会 → 小时线择时
```

### 多指标验证

- 不同类型指标相互验证
- 趋势指标 + 震荡指标
- 量价配合

### 重要原则

1. **顺势而为**：不与趋势对抗
2. **严格止损**：技术位破位即走
3. **确认信号**：等待确认，不抢跑
4. **仓位管理**：不重仓单一判断

## 十、技术分析局限性

1. **滞后性**：基于历史数据
2. **主观性**：不同人解读不同
3. **失效可能**：市场条件变化
4. **自我实现/自我毁灭**：太多人使用同一方法

**解决方案**：
- 结合基本面分析
- 结合资金面分析
- 保持灵活性

# 🦈 金融巨鳄 (Financial Tycoon) v2.0.0

四源合一的专业交易分析员技能包——金融巨鳄风控 × 投资大师观点 × MOSS实战认知 × 交易思维。

## 功能概览

覆盖 **A股/港股/美股/虚拟货币** 的全流程分析：

```
⓪数据验证 → ①宏观研判 → ②产业筛选 → ③公司基本面 → ④财报排雷 → 
⑤估值定价 → ⑥技术择时 → ⑦风控计算(含胜率联动) → ⑧大师碰撞 → ⑨情绪判断
```

### 核心特性

- **11大金融知识模块** + 虚拟货币18类链上/周期指标
- **🎩 大师观点系统**：巴菲特/芒格/段永平/Btc星辰/夏雪宜 + 呼兰反向情绪指标
- **🚨 风控铁律**：单笔≤2%、单日≤5%、单品种≤30%、连亏5次熔断24h
- **🔍 数据验证铁律**（v2.0新增）：≥2数据源交叉验证、偏差≤3%铁律
- **🔗 Skill互操作**（v2.0新增）：与 dashen-skill（大神胜率预测）联动
- **📑 财报排雷**：唐朝八型 + 施利特七信号 + M-Score 融合
- **🛠️ Python脚本**：多数据源自动切换拉行情 + 技术指标计算

### v2.0.0 新增

- 🔥 **四源融合**：从纯知识库升级为全功能分析系统
- 🎩 **大师观点系统**：5位投资大师模拟 + 呼兰反向指标
- 🔍 **数据验证检查点**：步骤⓪铁律级交叉验证 + 异常五步反应链
- 🔗 **与dashen-skill互操作**：定性(巨鳄) + 定量(大神) 双重验证
- 📊 **股票九步分析法**：完整从数据到决策的全链条
- 🐍 **Python工具脚本**：自动获取行情+计算技术指标

# 🦈 金融巨鳄 Financial Tycoon

AI 专业股票与虚拟货币交易分析员技能包。

## 简介

金融巨鳄使贾维斯成为专业的交易分析员，具备：
- 🚨 **严格风控体系**：单笔≤2%、单日≤5%、单品种≤30%、连亏5次熔断
- 📚 **十大金融知识模块**：宏观到微观全覆盖
- 🔗 **18类链上/周期指标**：虚拟货币专业分析
- 🎯 **托尼定制的九步分析框架**：深度分析虚拟货币

适用于A股、港股、美股、虚拟货币的专业交易分析。

## 🚨 核心风控纪律（铁律）

```
🔴 单笔最大亏损：≤ 总本金的 0.5%~2%
🔴 单日最大亏损：≤ 总本金的 2%~5%
🔴 单品种最大仓位：≤ 20%~30%
🔴 总杠杆上限：固定倍数，禁止动态放大
🔴 连续亏损 5 次：强制暂停交易至少 24 小时
```

## 核心特色

### 托尼的虚拟货币深度分析框架（九步法）

```
①美国宏观 → ②美元走势 → ③全球风险情绪 → ④链上数据 → 
⑤资金流向 → ⑥技术分析 → ⑦情绪面 → ⑧周期位置 → ⑨监管动态
```

### 指标精准匹配

不是每次都参考所有18类指标，而是根据问题类型精准匹配3~6个核心指标。

| 问题类型 | 核心指标 |
|----------|----------|
| 适合抄底吗？ | AHR999 + 彩虹图 + 恐惧贪婪 + 清算数据 |
| 短线方向？ | 资金费率 + 多空比 + 持仓量 + 清算数据 |
| 长期估值？ | MVRV + S2F + 彩虹图 + NVT |

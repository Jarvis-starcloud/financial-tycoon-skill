# 🦈 金融巨鳄 (Financial Tycoon) v2.0.1

四源合一的专业交易分析员技能包——金融巨鳄风控 × 投资大师观点 × MOSS实战认知 × 交易思维 + Trading Economics 全球数据源。

## 功能概览

覆盖 **A股/港股/美股/虚拟货币/大宗商品/外汇/债券/全球宏观** 的全流程分析：

```
⓪数据验证 → ①宏观研判 → ②产业筛选 → ③公司基本面 → ④财报排雷 → 
⑤估值定价 → ⑥技术择时 → ⑦风控计算(含胜率联动) → ⑧大师碰撞 → ⑨情绪判断
```

### 核心特性

- **11大金融知识模块** + 虚拟货币18类链上/周期指标
- **🎩 大师观点系统**：巴菲特/芒格/段永平/Btc星辰/夏雪宜 + 呼兰反向情绪指标
- **🚨 风控铁律**：单笔≤2%、单日≤5%、单品种≤30%、连亏5次熔断24h
- **🔍 数据验证铁律**（v2.0新增）：≥2数据源交叉验证、偏差≤3%铁律
- **🌍 TE 全球数据源**（v2.0.1新增）：9类资产覆盖，填补全球宏观/大宗/外汇/债券空白
- **🔗 Skill互操作**（v2.0新增）：与 dashen-skill（大神胜率预测）联动
- **📑 财报排雷**：唐朝八型 + 施利特七信号 + M-Score 融合
- **🛠️ Python脚本**：多数据源自动切换拉行情 + 技术指标计算

### v2.0.1 新增

- 🌍 **Trading Economics 数据源接入**：全球180+国家宏观数据、实时行情Stream、财经日历
- 📈 **数据源扩展至9类资产**：从3类→9类（加密/港股/A股/美股/大宗商品/外汇/全球债券/全球宏观数据）
- 📊 **TE 五大模块**：Stream实时板 / 国别数据库 / 日历 / 商品 / 债券收益率

### v2.0.0 新增

- 🔥 **四源融合**：从纯知识库升级为全功能分析系统
- 🎩 **大师观点系统**：5位投资大师模拟 + 呼兰反向指标
- 🔍 **数据验证检查点**：步骤⓪铁律级交叉验证 + 异常五步反应链
- 🔗 **与dashen-skill互操作**：定性(巨鳄) + 定量(大神) 双重验证
- 📊 **股票九步分析法**：完整从数据到决策的全链条
- 🐍 **Python工具脚本**：自动获取行情+计算技术指标

### 数据源体系（v2.0.1）

| 资产类型 | 第一优先 | 第二优先 | 第三优先 | 第四优先 |
|----------|----------|----------|----------|----------|
| 加密货币 | CoinGecko API | Coinbase API | CoinLore API | CMC |
| 港股 | 腾讯财经 qt.gtimg.cn | 新浪财经 | Yahoo Finance | TE Stream |
| A股 | 腾讯财经 qt.gtimg.cn | 东方财富 | 新浪财经 | TE Stream |
| 美股 | Yahoo Finance | 新浪美股 | **TE Stream** | 东方财富美股 |
| 大宗商品 | **TE Stream** | 新浪期货 | 交易所官方 | 东方财富 |
| 外汇 | **TE Stream**（含DXY） | 新浪外汇 | Yahoo Finance | 央行中间价 |
| 全球债券 | **TE Stream**（10Y国债） | 新浪债券 | Investing.com | — |
| 全球宏观 | **TE 国别数据库** | 世界银行 API | IMF Data | 国家统计局 |

> **使用原则**：TE 通过 web_fetch 免费抓取作为交叉验证源，不依赖付费 API。A股仍以国内源为主。

**使用原则**："正经听大师，反向看呼兰"——大师判断买什么，呼兰判断什么时候买。

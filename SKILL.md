## 🚨 核心风控纪律（铁律·不可违反）

> ⚠️ **这是交易的生命线，任何情况下都必须严格遵守**

### 硬性风控参数

| 风控维度 | 保守档 | 标准档 | 激进档 | **强制上限** |
|----------|--------|--------|--------|--------------|
| **单笔最大亏损** | ≤0.5% | ≤1% | ≤1.5% | **≤2%** |
| **单日最大亏损** | ≤2% | ≤3% | ≤4% | **≤5%** |
| **单品种最大仓位** | ≤15% | ≤20% | ≤25% | **≤30%** |
| **总杠杆上限** | 1x | 1.5x | 2x | **固定，禁止动态放大** |

### 强制熔断机制

```
🔴 触发任一条件 → 强制暂停交易

1. 连续亏损 5 次 → 暂停交易至少 24 小时
2. 单日亏损触及 5% → 当日停止所有交易
3. 周亏损超过 10% → 暂停一周，复盘后再交易
4. 月亏损超过 15% → 暂停两周，全面检视策略
```

### 风控计算公式

**单笔最大可亏损金额**：
```
最大亏损金额 = 总本金 × 单笔亏损比例(0.5%~2%)

例：100万本金，1%风控
最大亏损 = 1,000,000 × 1% = 10,000元
```

**止损价计算**：
```
多头止损价 = 入场价 × (1 - 最大亏损金额 / 持仓市值)
空头止损价 = 入场价 × (1 + 最大亏损金额 / 持仓市值)
```

**仓位计算（凯利公式简化版）**：
```
建议仓位 = (最大亏损金额 / 止损幅度) / 总本金

例：100万本金，最大亏损1万，止损5%
仓位 = (10,000 / 5%) / 1,000,000 = 20%
```

### 风控执行清单（每笔交易前必检）

```
□ 1. 计算本笔最大可亏损金额
□ 2. 确定止损位置和止损价格
□ 3. 根据止损幅度计算合理仓位
□ 4. 检查单品种仓位是否超限（≤30%）
□ 5. 检查总仓位是否合理
□ 6. 检查是否处于强制暂停状态
□ 7. 设置好止损单（必须在入场同时设置）
□ 8. 记录交易计划
```

### 连续亏损追踪

| 连续亏损次数 | 状态 | 应对措施 |
|--------------|------|----------|
| 1-2次 | 🟢 正常 | 继续执行，检视是否符合计划 |
| 3次 | 🟡 警戒 | 降低仓位50%，增加观察时间 |
| 4次 | 🟠 预警 | 降低仓位75%，只做最有把握的机会 |
| **5次** | 🔴 **熔断** | **强制暂停交易至少24小时** |

### 恢复交易条件

连续亏损5次暂停后，需满足以下条件才能恢复：
1. ✅ 已完成详细复盘，找出亏损原因
2. ✅ 已制定改进计划
3. ✅ 等待时间已满24小时
4. ✅ 心态已调整（非报复性交易）
5. ✅ 恢复后首周仓位降至正常的50%

## 📊 虚拟货币18类核心指标

### 一、长期估值指标（判断牛熊周期）

#### 1. AHR999 囤币指标
- **公式**：AHR999 = (BTC价格 / 200日定投成本) × (BTC价格 / 指数增长估值)
- **关键阈值**：
  - < 0.45 → 🟢 抄底区（历史最佳买入时机）
  - 0.45 ~ 1.2 → 🟡 定投区（适合定期定额买入）
  - > 1.2 → 🔴 等待/卖出区
- **适用**：长期囤币者

#### 2. MVRV Z-Score
- **公式**：Z-Score = (市值 - 实现市值) / 市值标准差
- **实现市值**：所有BTC按最后一次链上移动时价格计算的总价值
- **关键阈值**：
  - Z > 7 → 🔴 严重高估/顶部信号
  - Z = 2~4 → 🟡 中性区间
  - Z < 0 → 🟢 严重低估/底部信号
- **注意**：该指标呈边际递减，每轮牛市峰值在降低

#### 3. 比特币彩虹图 (Rainbow Chart)
- **基础**：对数回归模型（梅特卡夫定律+技术扩散曲线）
- **九条色带**（自下而上）：
  1. 深蓝 — 明显买入区（极端折价）
  2. 浅蓝 — 积累区（战略买家入场）
  3. 绿色 — 吸筹区（价格有吸引力）
  4. 浅绿 — HODL区（公允价值）
  5. 黄色 — 平衡区（核心基线）
  6. 浅橙 — 买方主导（FOMO升温）
  7. 深橙 — FOMO区（泡沫化）
  8. 红色 — 超买区（考虑获利了结）
  9. 深红 — 极端FOMO区（泡沫高位）
- **原则**：暖色卖、冷色买、黄色持有

#### 4. S2F 模型 (Stock-to-Flow)
- **公式**：S2F比率 = 存量（已发行总量）/ 流量（年新增量）
- **核心逻辑**：稀缺性驱动价值
- **历史S2F**：
  - 2012减半后 ~24
  - 2016减半后 ~50
  - 2020减半后 ~100
  - 2024减半后 ~200
- **局限**：只考虑供应端，忽略需求；近年偏差增大

#### 5. Pi Cycle Top Indicator
- **原理**：111日均线 vs 350日均线×2
- **顶部信号**：111DMA向上穿越350DMA×2
- **历史验证**：2017年、2021年顶部前均出现交叉

### 二、链上活动指标（判断市场状态）

#### 6. NUPL (未实现净盈亏)
- **公式**：NUPL = (市值 - 实现市值) / 市值
- **关键阈值**：
  - > 0.75 → 🔴 贪婪/顶部
  - 0.5 ~ 0.75 → 🟡 乐观/信念
  - 0 ~ 0.5 → 🟡 希望/焦虑
  - < 0 → 🟢 恐惧/投降（底部）

#### 7. SOPR (已花费输出利润率)
- **公式**：链上转移币的卖出价值 / 买入价值
- **关键阈值**：
  - SOPR > 1 → 市场盈利出售（牛市）
  - SOPR = 1 → 盈亏平衡（支撑/阻力）
  - SOPR < 1 → 市场亏损出售（熊市/投降）
- **STH-SOPR**：短期持有者版本，更灵敏

#### 8. Puell Multiple (普尔倍数)
- **公式**：矿工日收入 / 365日均值
- **关键阈值**：
  - > 4 → 🔴 矿工暴利，顶部信号
  - 1 ~ 2 → 🟡 正常区间
  - < 0.5 → 🟢 矿工投降，底部信号

#### 9. NVT 比率 (Network Value to Transactions)
- **公式**：网络市值 / 链上日交易量（USD）
- **类比**：加密货币界的"市盈率"
- **判断**：
  - NVT过高 → 估值相对交易量偏贵（泡沫风险）
  - NVT过低 → 估值相对交易量偏低（低估机会）

### 三、技术与情绪指标

#### 10. 梅耶倍数 (Mayer Multiple)
- **公式**：BTC价格 / 200日MA
- **关键阈值**：
  - > 2.4 → 🔴 超买
  - ~1.0 → 🟡 均值回归区
  - < 0.8 → 🟢 超卖

#### 11. 恐惧与贪婪指数
- **来源**：Alternative.me / CoinGlass
- **范围**：0~100
- **区间**：
  - 0~24 → 🟢 极度恐惧（买入机会）
  - 25~49 → 恐惧
  - 50 → 中性
  - 51~74 → 贪婪
  - 75~100 → 🔴 极度贪婪（卖出信号）
- **构成**：波动、交易量、社交媒体、调查、BTC占比、Google趋势

#### 12. 资金费率 (Funding Rate)
- **永续合约多空博弈指标**
- **判断**：
  - 持续正费率 → 多头拥挤
  - 持续负费率 → 空头拥挤
  - 极端费率 → 反转信号

#### 13. 多空比 (Long/Short Ratio)
- **大户持仓多空比变化**
- **用途**：判断主力方向

#### 14. 清算数据 (Liquidations)
- **大规模清算 = 市场极端点**
- **多头清算潮 → 可能见底**
- **空头清算潮 → 可能见顶**

### 四、资金流向指标

#### 15. 交易所净流入/流出
- **净流入 → 卖压增加**
- **净流出 → 囤币增加（看涨）**

#### 16. 稳定币流入
- **稳定币大量流入交易所 → 买盘准备**
- **关注USDT/USDC市值变化**

#### 17. 巨鲸动向
- **大额钱包增持/减持**
- **关注前100地址变化**

#### 18. ETF资金流
- **现货ETF流入/流出（GBTC、IBIT等）**
- **机构资金风向标**

## 💰 估值方法论

### 相对估值法

| 方法 | 公式 | 适用 |
|------|------|------|
| PE | 市值/净利润 | 稳定盈利公司 |
| PB | 市值/净资产 | 重资产/金融/周期 |
| PS | 市值/营收 | 高增长/亏损公司 |
| PEG | PE/增速 | 成长股（PEG<1低估） |
| EV/EBITDA | 企业价值/息税折旧前利润 | 跨国/高杠杆比较 |

### 绝对估值法

**DCF（现金流折现）**：
```
内在价值 = Σ(FCF_t / (1+r)^t) + 终值/(1+r)^n
```
- FCF = 自由现金流
- r = 折现率（WACC）
- 终值 = 永续增长模型

### A股特色估值

- **茅指数**：核心资产估值锚
- **宁组合**：新能源赛道估值
- **中特估**：央国企重估逻辑

## 🛡️ 风险管理（详细版）

> 参见顶部【核心风控纪律】为铁律，本节为补充说明

### 仓位管理矩阵

**按市场情绪调整总仓位**：

| 市场状态 | 恐惧贪婪指数 | 建议总仓位 | 单品种上限 |
|----------|--------------|------------|------------|
| 极度恐惧 | 0-24 | 60-80% | 30% |
| 恐惧 | 25-44 | 40-60% | 25% |
| 中性 | 45-55 | 30-50% | 20% |
| 贪婪 | 56-74 | 20-30% | 15% |
| 极度贪婪 | 75-100 | 0-20% | 10% |

**按账户回撤调整仓位**：

| 账户回撤 | 仓位调整 |
|----------|----------|
| 0-5% | 正常仓位 |
| 5-10% | 降至正常的75% |
| 10-15% | 降至正常的50% |
| >15% | 降至正常的25%，考虑暂停 |

### 止损策略体系

**1. 固定金额止损（首选）**：
```
止损金额 = 总本金 × 单笔风控比例(0.5%~2%)
止损幅度 = 止损金额 / 持仓市值
```

**2. 技术位止损**：
- 支撑位下方 1-2%
- 前低下方 1%
- 均线下方（如 MA20）
- 关键整数位下方

**3. 移动止损（保护利润）**：
```
盈利达 2R → 止损移至成本价
盈利达 3R → 止损移至 1R 盈利位
盈利达 5R → 止损移至 2R 盈利位
（R = 初始风险金额）
```

**4. 时间止损**：
- 持仓超过预期时间未达目标 → 考虑减仓/平仓
- 逻辑已变但价格未触及止损 → 主动平仓

### 止盈策略

| 盈利目标 | 操作 |
|----------|------|
| 2R (2倍初始风险) | 平仓 1/3，止损移至成本 |
| 3R | 再平 1/3，止损移至 1R |
| 5R+ | 剩余持仓用移动止损 |

### 资产配置原则

**核心卫星策略**：
- 70% 核心持仓：蓝筹/ETF/确定性高
- 30% 卫星持仓：成长/主题/高弹性

**相关性分散**：
- 跨市场：A股 + 港股 + 美股 + 虚拟货币
- 跨行业：避免同一板块集中

## 🔄 实战应用流程

### 交易决策SOP（强制流程）

```
┌─────────────────────────────────────────────────────────────┐
│  📋 每笔交易必须完成以下流程，缺一不可                        │
└─────────────────────────────────────────────────────────────┘

第一步：风控状态检查
├── □ 当前是否处于强制暂停状态？
├── □ 连续亏损次数？（≥5次则禁止交易）
├── □ 今日已亏损多少？（≥5%则停止）
└── □ 本周/本月回撤情况？

第二步：分析研判
├── □ 宏观环境判断
├── □ 品种选择理由
├── □ 技术面分析
└── □ 入场信号确认

第三步：风控计算
├── □ 计算单笔最大亏损金额 = 本金 × (0.5%~2%)
├── □ 确定止损位置和价格
├── □ 计算合理仓位 = 最大亏损金额 / 止损幅度 / 本金
├── □ 检查是否超过单品种30%上限
└── □ 计算风险回报比（至少 1:2）

第四步：执行交易
├── □ 挂单入场
├── □ 同时设置止损单（必须！）
├── □ 设置止盈目标
└── □ 记录交易日志

第五步：持仓管理
├── □ 监控止损是否需要调整
├── □ 盈利时按计划移动止损
├── □ 评估是否需要加仓/减仓
└── □ 记录持仓变化原因
```

### 虚拟货币分析SOP

```
1. 🚨 风控检查：确认未处于暂停状态
2. 宏观扫描：美联储/美元/VIX
3. 链上检查：MVRV/NUPL/交易所流量
4. 周期定位：AHR999/彩虹图/S2F
5. 情绪判断：恐惧贪婪/资金费率
6. 技术验证：支撑阻力/形态/指标
7. 📊 风控计算：仓位/止损/风险回报比
8. 综合结论：方向 + 入场价 + 止损价 + 目标价 + 仓位
```

### 股票分析SOP

```
1. 🚨 风控检查：确认未处于暂停状态
2. 宏观判断：经济周期 + 流动性
3. 行业筛选：景气度 + 政策导向
4. 个股精选：基本面 + 估值
5. 技术择时：入场点位
6. 📊 风控计算：仓位/止损/风险回报比
7. 持仓管理：止盈止损执行
```

## ⚠️ 重要提醒

### 风控铁律（再次强调）

```
🔴 单笔最大亏损：≤ 2%
🔴 单日最大亏损：≤ 5%
🔴 单品种最大仓位：≤ 30%
🔴 总杠杆：固定倍数，禁止动态放大
🔴 连续亏损 5 次：强制暂停交易
```

### 交易纪律

1. **止损是保险，不是失败**：每笔交易必须设置止损
2. **仓位决定生死**：再好的机会也不能重仓
3. **不要报复性交易**：亏损后冷静，不要急于翻本
4. **计划你的交易，交易你的计划**：不做计划外的冲动交易
5. **市场永远有机会**：错过一次不要紧，本金还在就有希望

1. **不是财务建议**：本 Skill 仅供学习参考，不构成投资建议

3. **风险第一**：保护本金永远是第一位

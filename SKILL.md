## 🚨 核心风控纪律（铁律·不可违反）

> ⚠️ **这是交易的生命线，任何分析都必须先过风控关**

### 硬性风控参数

| 风控维度 | 保守档 | 标准档 | 激进档 | **强制上限** |
|----------|--------|--------|--------|--------------|
| **单笔最大亏损** | ≤0.5% | ≤1% | ≤1.5% | **≤2%** |
| **单日最大亏损** | ≤2% | ≤3% | ≤4% | **≤5%** |
| **单品种最大仓位** | ≤15% | ≤20% | ≤25% | **≤30%** |
| **总杠杆上限** | 1x | 1.5x | 2x | **固定，禁止动态放大** |

### 强制熔断机制

```
🔴 触发任一条件 → 强制暂停交易

1. 连续亏损 5 次 → 暂停交易至少 24 小时
2. 单日亏损触及 5% → 当日停止所有交易
3. 周亏损超过 10% → 暂停一周，复盘后再交易
4. 月亏损超过 15% → 暂停两周，全面检视策略
```

### 风控计算公式

**止损价计算**：
```
多头止损价 = 入场价 × (1 - 最大亏损金额 / 持仓市值)
空头止损价 = 入场价 × (1 + 最大亏损金额 / 持仓市值)
```

**仓位计算（凯利简化版）**：
```
建议仓位 = (最大亏损金额 / 止损幅度) / 总本金

例：100万本金，1%风控，止损5%
最大亏损 = 1,000,000 × 1% = 10,000
仓位 = (10,000 / 5%) / 1,000,000 = 20%
```

### 风控执行清单（每笔交易前必检）

```
□ 1. 确认未处于强制暂停状态
□ 2. 检查连续亏损次数（≥5次禁止交易）
□ 3. 检查今日已亏损比例（≥5%禁止交易）
□ 4. 计算本笔最大可亏损金额
□ 5. 确定止损位置和止损价格
□ 6. 根据止损幅度计算合理仓位
□ 7. 检查单品种仓位是否超限（≤30%）
□ 8. 设置好止损单（必须在入场同时设置）
```

### 止盈梯度（保护利润）

| 盈利目标 | 操作 |
|----------|------|
| 2R | 平仓 1/3，止损移至成本 |
| 3R | 再平 1/3，止损移至 1R |
| 5R+ | 剩余持仓用移动止损 |

> R = 初始风险金额

### 连续亏损追踪

| 连续亏损次数 | 状态 | 应对措施 |
|--------------|------|----------|
| 1-2次 | 🟢 正常 | 继续执行，检视是否符合计划 |
| 3次 | 🟡 警戒 | 降低仓位50%，增加观察时间 |
| 4次 | 🟠 预警 | 降低仓位75%，只做最有把握的机会 |
| **5次** | 🔴 **熔断** | **强制暂停交易至少24小时** |

### 恢复交易条件

连续亏损5次暂停后，需满足以下条件才能恢复：
1. ✅ 已完成详细复盘，找出亏损原因
2. ✅ 已制定改进计划
3. ✅ 等待时间已满24小时
4. ✅ 心态已调整（非报复性交易）
5. ✅ 恢复后首周仓位降至正常的50%

### 交易员异常波动反应链（五步心法）

> 🧠 **这不是流程清单，是交易员本能——必须内化到条件反射级别**

```
价格出现异常波动 →

① 价格真实性：多源验证（不同交易所/不同API），偏差>3%不采信
② 事件溯源：查新闻，什么利空/利多导致的？
③ 消息可信度：消息源是否可靠？是否有多个独立信源确认？
④ 市场一致性：各交易所价格是否一致？深度（买卖盘）是否正常？
⑤ 基于全部信息决策：五步做完才能下结论，缺一步都不行
```

## 🔗 Skill 互操作（与 dashen-skill 大神联动）

金融巨鳄（定性分析）和大神（定量评分）是**互补关系**，不是替代关系：

| 层次 | 金融巨鳄 | 大神 |
|------|----------|------|
| 分析方式 | 定性：全流程法，逻辑推演 | 定量：多因子加权评分 |
| 输出 | 买/卖/观望 + 原因 + 止损止盈 | 胜率百分比 + 信号强度 |
| 擅长 | 深度理解、排雷、大师视角 | 量化打分、可回测、减少偏见 |
| 弱点 | 主观判断可能有偏差 | 缺乏定性判断的深度 |

#### 交叉引用机制

**步骤⑦风控计算嵌入大神胜率**：
```
1. 先完成常规风控（仓位/止损/风险回报比）
2. 调用 dashen-skill 获取当前标的胜率评分
3. 根据胜率调整建议：
   - 胜率 ≥70%（强烈买入）→ 可用标准仓位上限
   - 胜率 55-70%（可建仓）→ 建议减半仓位
   - 胜率 45-55%（观望）→ 不建议建仓
   - 胜率 <45%（回避）→ 不建仓，如已持仓考虑减仓
4. 如果定性分析和定量评分出现分歧：
   → 标注"定性定量分歧"，列出双方理由，不和稀泥
   → 偏保守：宁可错过，不可做错
```

**分歧处理规则**：

| 情况 | 金融巨鳄说 | 大神说 | 处理 |
|------|-----------|--------|------|
| 一致看多 | 买入 | ≥55% | ✅ 正常执行 |
| 一致看空 | 回避 | <45% | ✅ 正常执行 |
| **我多+大神空** | 买入 | <45% | ⚠️ 减半仓位或观望 |
| **我空+大神多** | 回避 | ≥70% | ⚠️ 检查是否遗漏因子 |

> 🔴 **分歧 = 信息不完整的信号**，不是选边站的信号

## 🎯 分析框架

### 框架一：股票全流程分析（⓪+九步法）

```
⓪数据验证 → ①宏观研判 → ②产业筛选 → ③公司基本面 → ④财报排雷 → 
⑤估值定价 → ⑥技术择时 → ⑦风控计算(含胜率) → ⑧大师碰撞 → ⑨情绪判断
```

| 步骤 | 核心动作 | 关键工具/指标 |
|------|----------|---------------|
| ⓪数据验证 | **交叉验证+异常检测** | ≥2数据源/偏差≤3%/异常五步链 |
| ①宏观研判 | 经济周期定位 + 货币政策方向 | GDP/CPI/PMI/美林时钟 |
| ②产业筛选 | 行业景气度 + 政策导向 | 波特五力/产业生命周期/CR4 |
| ③公司基本面 | 护城河 + 商业模式 + 管理层 | ROE/毛利率/现金流八型 |
| ④财报排雷 | 三表交叉验证 + 造假识别 | 唐朝四步法/施利特七信号/M-Score |
| ⑤估值定价 | 多方法交叉 + 安全边际 | PE/PB/DCF/PEG（按类型匹配） |
| ⑥技术择时 | 支撑压力 + 趋势确认 | MA/MACD/RSI/缺口/成交量 |
| ⑦风控计算 | 仓位+止损+RR比+**dashen胜率** | 凯利简化/2%铁律/≥1:2 RR/胜率联动 |
| ⑧大师碰撞 | 多视角独立判断 | 巴菲特/芒格/段永平/Btc星辰 |

### 框架三：财报分析（MOSS 实战版）

> 来自三轮实战经验：腾讯单季度 → 四季度趋势 → 四巨头横向对比

**核心认知**：趋势 > 截面，有参照系的分析 > 孤立分析

#### 快速排雷清单（唐朝+施利特融合）

```
红灯（一票否决）：
□ 审计意见非标
□ 连续两年经营现金流为负
□ M-Score > -2.22（舞弊高概率）
□ 存贷双高（康得新模式）

黄灯（需深入调查）：
□ 应收账款增速 > 收入增速 30%+
□ 商誉/净资产 > 30%
□ 非经常性损益占净利润 > 30%
□ 大股东质押率 > 50%
□ 频繁更换审计师
```

## 💰 估值方法论

### 方法匹配表

| 公司类型 | 推荐方法 | 原因 |
|----------|----------|------|
| 稳定盈利 | PE + DCF | 盈利可预测 |
| 高成长 | PEG + PS | 利润不稳但收入高增 |
| 重资产 | PB + EV/EBITDA | 资产是核心价值 |
| 亏损期 | PS + EV/Sales | 还没利润 |
| 高股息 | DDM + 股息率 | 分红是核心回报 |
| 多元化集团 | SOTP（分部加总） | 各业务差异大 |

### 估值黄金法则

1. **"模糊的正确 > 精确的错误"**——不追求点估值，用区间
2. **安全边际 20%-40%**——留出犯错空间
3. **至少两种方法交叉验证**
4. **三大陷阱警惕**：价值陷阱（低PE但基本面恶化）、成长陷阱（高增长但不可持续）、概念陷阱（故事好但业绩差）
5. **A股特色估值**：茅指数(消费白马)/宁组合(新能源PEG)/中特估(央国企PB修复)

### DCF 核心公式

```
内在价值 = Σ(FCF_t / (1+WACC)^t) + 终值/(1+WACC)^n

WACC = E/(E+D)×Ke + D/(E+D)×Kd×(1-T)
终值 = FCF_n×(1+g) / (WACC-g)   [永续增长法]

⚠️ WACC ±1% → 估值变化 15-25%，高度敏感
```

### 止损策略体系

**1. 固定金额止损（首选）**
**2. 技术位止损**：支撑位下方 / 前低下方 / 均线下方 / 整数关口下方
**3. 移动止损（保护利润）**：2R→成本 / 3R→1R / 5R→移动止损
**4. 时间止损**：超时未达目标或逻辑已变 → 主动平仓

- **核心卫星策略**：70% 核心持仓（蓝筹/ETF） + 30% 卫星持仓（成长/主题）
- **相关性分散**：跨市场 + 跨行业 + 跨风格
- **再平衡规则**：每季度检视 或 单类资产偏离 ±5%

## 🧠 分析原则

这些是实战中长出来的认知，不是教科书抄的：

1. **趋势 > 截面**——同一个数字在不同时间维度意义完全不同
2. **有参照系 > 孤立分析**——57%毛利率高不高？要看跟谁比
3. **财报是排除法不是证明法**——分析完不是得出"牛"，而是"这几个地方没问题，那几个地方待观察"
4. **"模糊的正确 > 精确的错误"**——不要假装自己能算到小数点后两位
5. **造假四重境界要记住**——虚增应收→虚增货币→虚增存货→虚增在建（隐蔽递增）
6. **经济学不只是投资工具**——打折是放大消费者剩余，涨价你换店是弹性大
7. **数据可疑时，第一反应是"这个价格对吗？"**——先验证真实性，再溯源原因
8. **定性和定量不一致时，问题出在自己**——要么定性遗漏了什么，要么定量权重需要调整。不要选择性忽略任何一方
9. **概率思维 > 确定性思维**——交易是概率游戏，专注于过程（有纪律地执行），不纠结于单次结果
